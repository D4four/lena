セットアップの必要条件
CorelDRAW Graphics Suite X6.1 をインストールするには、オペレーティング システムの一部のコンポーネントをアップデートする必要があります。 [続行] をクリックして次のコンポーネントをインストールするか、 [キャンセル] をクリックしてセットアップを終了してください。
必要なコンポーネント:

Microsoft .NET Framework 3.5 Service Pack 1
Microsoft .NET Framework 3.5 Service Pack 1 (注意: このアップデートにはインターネット接続が必要です。)
Microsoft .NET Framework 4
Microsoft .NET Framework 4 (注意: このアップデートにはインターネット接続が必要です。)
Microsoft Windows Installer 4.5
