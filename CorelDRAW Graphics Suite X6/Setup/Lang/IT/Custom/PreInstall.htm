Prerequisiti per l’installazione
Per poter installare CorelDRAW Graphics Suite X6.1, è necessario aggiornare alcuni componenti del sistema operativo. Fare clic su Continua per installare i componenti seguenti oppure fare clic su Annulla per uscire dall’installazione.
Componenti necessari:

Microsoft .NET Framework 3.5 Service Pack 1
Microsoft .NET Framework 3.5 Service Pack 1 (NOTA: questo aggiornamento richiede una connessione a Internet).
Microsoft .NET Framework 4
Microsoft .NET Framework 4 (NOTA: questo aggiornamento richiede una connessione a Internet).
Microsoft Windows Installer 4.5
