Conditions requises pour l’installation
Certains composants de votre système d’exploitation doivent être mis à jour avant d’installer CorelDRAW Graphics Suite X6.1. Cliquez sur Continuer pour installer les composants suivants ou cliquez sur Annuler pour quitter l’assistant d’installation.
Composants requis :

Microsoft .NET Framework 3.5 Service Pack 1
Microsoft .NET Framework 3.5 Service Pack 1 (REMARQUE : cette mise à jour nécessite une connexion Internet.)
Microsoft .NET Framework 4
Microsoft .NET Framework 4 (REMARQUE : cette mise à jour nécessite une connexion Internet.)
Microsoft Windows Installer 4.5
