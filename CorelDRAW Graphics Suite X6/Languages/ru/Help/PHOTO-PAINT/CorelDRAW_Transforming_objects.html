Преобразование объектов

Вид объектов можно изменить с помощью следующих преобразований.

Преобразование
Описание
Размер
Изменение ширины и высоты объекта
Масштаб
Изменение размера объекта в процентном отношении к его исходному размеру
Поворот
Поворот объекта вокруг его центра вращения.
Переворот
Создание горизонтального или вертикального зеркального изображения объекта
Наклон
Наклон объекта в сторону
Искажение
Растяжение объекта с нарушением пропорций
Применение перспективы
Придание объекту эффекта глубины

В окне изображения можно применить преобразования свободной формы или вручную настроить параметры для достижения более точных результатов.

Преобразования можно применить для одного или нескольких объектов одновременно.

Преобразование
Применение для объектов на изображении

Изменение размера и масштабирование

Масштаб объекта на фотографии уменьшается в соответствии с размером фонового изображения.

Переворот

Объект транспонируется для создания эффекта отражения.

Поворот

Отражение поворачивается.

Наклон

Отражение наклоняется для создания реалистичного угла.

Искажение

Тень искажается, указывая направление источника света.

Перспектива

Добавляется и модифицируется вторая тень.

Применение перспективы для объекта
Изменение размера объекта
1.
 
Выделите объект.
2.
 
Нажмите кнопку Положение и размеры на панели свойств.
3.
 
Перетащите любой из маркеров ограничивающего блока.
Для отмены преобразования дважды щелкните за пределами объекта.
4.
 
Нажмите кнопку Применить преобразование на панели свойств.
Можно изменить размер объекта от центра, удерживая нажатой клавишу Shift при перетаскивании любого маркера.
Для изменения размера выбранного изображения можно также ввести значения в полях Размер и нажать кнопку Применить преобразование на панели свойств.
Масштабирование объекта
1.
 
Выделите объект.
2.
 
Нажмите кнопку Масштаб на панели свойств.
3.
 
Перетащите угловой маркер ограничивающего блока.
Для отмены преобразования дважды щелкните за пределами объекта.
4.
 
Нажмите кнопку Применить преобразование на панели свойств.
Для масштабирования выбранного объекта можно также ввести значения в полях Размер и нажать кнопку Применить преобразование на панели свойств.
После масштабирования, наклона или поворота объекта его края могут стать неровными. Поэтому для данных режимов преобразования по умолчанию включена функция сглаживания.
Поворот объекта
1.
 
Выделите объект.
2.
 
Нажмите кнопку Поворот на панели свойств.
3.
 
Перетащите маркер поворота ограничивающего блока.
Для отмены преобразования дважды щелкните за пределами объекта.
4.
 
Нажмите кнопку Применить преобразование на панели свойств.

Дополнительные возможности
 
Сдвиг и поворот объекта
Введите значения в полях Положение и Угол поворота на панели свойств и нажмите кнопку Применить преобразование .
Ограничьте поворот, задав для перемещения значение 15°
Во время перетаскивания углового маркера удерживайте нажатой клавишу Ctrl.
Изменение точки поворота
Перетащите центр вращения объекта в новое положение.

Для переключения в режим поворота можно также дважды щелкнуть объект. При переходе из одного режима в другой маркеры, окружающие объект, будут изменяться, указывая на активный режим.
После масштабирования, наклона или поворота объекта его края могут стать неровными. Поэтому для данных режимов преобразования по умолчанию включена функция сглаживания.
Переворот объекта
1.
 
Выделите объект.
2.
 
Удерживая нажатой клавишу Ctrl, перетащите средний маркер на поле выделения через объект за средний маркер на противоположной стороне.
Для отмены преобразования дважды щелкните за пределами объекта.
3.
 
Нажмите кнопку Применить преобразование на панели свойств.
Выбранный объект можно перевернуть с нарушением пропорций, перетащив средний маркер поля выделения через средний маркер противоположной стороны, не нажимая при этом клавишу Ctrl.
Наклон объекта
1.
 
Выделите объект.
2.
 
Нажмите кнопку Наклон на панели свойств.
3.
 
Перетащите маркер наклона ограничивающего блока.
Для отмены преобразования дважды щелкните за пределами объекта.
4.
 
Нажмите кнопку Применить преобразование на панели свойств.
После масштабирования, наклона или поворота объекта его края могут стать неровными. Поэтому для данных режимов преобразования по умолчанию включена функция сглаживания.
Для наклона объекта можно также ввести значения в полях Размер и нажать кнопку Применить преобразование на панели свойств.
Искажение объекта
1.
 
Выделите объект.
2.
 
Нажмите кнопку Искажение на панели свойств.
3.
 
Перетащите маркер искажения ограничивающего блока.
Для отмены преобразования дважды щелкните за пределами объекта.
Для переключения в режим искажения можно также дважды щелкнуть объект. При переходе из одного режима в другой маркеры, окружающие объект, будут изменяться, указывая на активный режим.
Применение перспективы для объекта
1.
 
Выделите объект.
2.
 
Нажмите кнопку Перспектива на панели свойств.
3.
 
Перетащите маркер перспективы ограничивающего блока.
Для отмены преобразования дважды щелкните за пределами объекта.
Для переключения в режим перспективы можно также щелкнуть объект три раза. При переходе из одного режима в другой маркеры, окружающие объект, будут изменяться, указывая на активный режим.
