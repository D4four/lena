Метафайл Windows (WMF)

Данный формат файла, разработанный корпорацией Microsoft, предназначен для хранения изображений как в векторном, так и в растровом формате. Формат WMF был разработан как собственный формат Microsoft Windows 3. Он поддерживает цветовую палитру RGB (24-бита) и распознается большинством приложений Windows.
