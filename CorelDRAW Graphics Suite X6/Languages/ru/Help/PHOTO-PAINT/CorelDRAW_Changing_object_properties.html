Изменение свойств объектов

Можно переименовать объект и изменить его свойства. При создании объекта ему задается имя по умолчанию, например Объект 2. Если специальные параметры не заданы, для объекта применяются параметры по умолчанию. В некоторых программах свойства объекта также называются параметрами слоя.

Можно изменить непрозрачность объекта, выбрать режим слияния, а также изменить способ перетекания объекта в расположенные под ним объекты или на изображение фона. Для получения дополнительных сведений о режимах слияния см. раздел Общие сведения о режимах слияния.

Изменяя свойства объекта, можно также превратить объект в активную область карты изображения для Интернета. Для получения дополнительных сведений см. раздел Создание карт изображений.

Изменение свойств объекта
1.
 
В наборе инструментов выберите инструмент Выбор объекта .
2.
 
Выберите в окне изображения объект.
3.
 
Щелкните объект правой кнопкой мыши и выберите Свойства объекта.
4.
 
Перейдите на вкладку Общие.
5.
 
Введите новое имя в поле Имя.

Дополнительные возможности
 
Изменение непрозрачности объекта.
Переместите регулятор Непрозрачность.
Выбор режима слияния
Выберите режим слияния в списке Режим слияния.
Изменение способа перетекания объекта в расположенные под ним объекты или в фон
Измените параметры в области Перетекание.

Имя объекта не может содержать более 39 символов.
Объект можно также переименовать, дважды щелкнув его имя в окне настройки Объекты.
Режим слияния для группы объектов можно выбрать в списке Режим слияния окна настройки Объекты.
