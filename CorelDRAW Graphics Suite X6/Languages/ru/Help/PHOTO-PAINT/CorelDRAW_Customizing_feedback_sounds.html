Настройка звуковых сигналов извещения

Данное приложение поддерживает воспроизведение звуков операционной системы Windows. Эти звуки, представляющие собой сигналы извещения посредством аудиосигнала, связаны с определенными событиями интерфейса пользователя. Несмотря на то что данное приложение регистрирует события, доступные для аудиосигналов, в нем не используются никакие звуки. Для определенных событий можно задать звуковые сигналы извещения. Если звуковые сигналы извещения назначены, они будут включены по умолчанию, однако их можно и отключить.

Назначение звуковых сигналов извещения (Windows 7 и Windows Vista)
1.
 
На панели задач Windows нажмите кнопку Пуск Панель управления.
2.
 
Выберите пункт Оборудование и звук.
3.
 
В разделе Звук нажмите кнопку Изменение системных звуков.
Откроется диалоговое окно Звук.
4.
 
В списке Программные события выберите событие под названием приложения.
5.
 
Выберите звуковой эффект в списке Звуки.
Дополнительные сведения о настройке звуковых сигналов извещения см. в справке Windows.
Назначение звуковых сигналов извещения (Windows XP)
1.
 
На панели задач Windows нажмите кнопку Пуск.
2.
 
Щелкните Панель управления.
3.
 
Выберите Звук, речь и аудиоустройства.
4.
 
Выберите Звуки и аудиоустройства.
5.
 
В диалоговом окне Звуки и свойства звука перейдите на вкладку Звуки.
6.
 
В списке Программные события выберите событие ниже названия приложения.
7.
 
Выберите звуковой эффект в списке Звуки.
Отключение звуковых сигналов извещения
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий Рабочее пространство выберите Общие.
3.
 
Снимите флажок Включить звуковые эффекты.
Чтобы активизировать команду отключения, требуется перезапустить приложение.
