Повтор мазков кисти

Мазок кисти можно сохранить и повторно нанести его на то же или другое изображение. Мазок кисти можно также применить вдоль границы пути или маски. Для получения сведений о нанесении мазка кисти на путь см. раздел Нанесение мазков кисти на пути.

Сохраненный мазок кисти можно отредактировать, чтобы создать новые эффекты путем настройки таких атрибутов, как размер, количество, угол и цвет мазка кисти.

Сохранение мазка кисти
1.
 
В наборе инструментов выберите инструмент Краска .
2.
 
Откройте на панели свойств меню выбора Категория кисти и выберите инструмент.
3.
 
Выберите мазок кисти в списке Тип кисти.
4.
 
Выберите Правка Повторить мазок кисти.
5.
 
В диалоговом окне Повтор мазка щелкните раскрывающую стрелку Мазок и выберите Добавить последний мазок.
6.
 
Выберите папку, в которой требуется сохранить мазок кисти.
7.
 
Введите имя файла в поле Имя файла.
Нанесение сохраненного мазка кисти
1.
 
В наборе инструментов выберите инструмент Краска .
2.
 
Откройте на панели свойств меню выбора Категория кисти и выберите инструмент.
3.
 
Выберите Правка Повторить мазок кисти.
Если имеются два элемента меню с именем Повторить мазок кисти, выберите второй.
4.
 
Выберите мазок кисти в списке Мазок.
5.
 
Щелкните в окне изображения, чтобы нанести мазок кисти.
Если требуется нанести несколько мазков кисти, щелкните несколько раз.
Редактирование сохраненного мазка кисти
1.
 
В наборе инструментов выберите инструмент Краска .
2.
 
Откройте на панели свойств меню выбора Категория кисти и выберите инструмент.
3.
 
Выберите Правка Повторить мазок кисти.
Если имеются два элемента меню с именем Повторить мазок кисти, выберите второй.
4.
 
В диалоговом окне Повтор мазка выберите сохраненный мазок кисти в списке Мазок.
5.
 
В диалоговом окне Повтор мазка измените любые атрибуты.
6.
 
Щелкните в окне изображения, чтобы нанести мазок кисти.
