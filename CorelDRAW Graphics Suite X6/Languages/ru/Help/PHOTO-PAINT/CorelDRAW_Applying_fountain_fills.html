Применение фонтанных заливок

Фонтанная заливка постепенно изменяет цвет на следующий, идущий вдоль линейного, радиального, конического, прямого или прямоугольного пути. Фонтанные заливки можно использовать для создания иллюзии глубины. Можно выбрать заготовку заливки или создать двухцветную или специальную фонтанную заливку.

Чтобы создать специальную фонтанную заливку, можно изменить цвета, настроить центральную точку или изменить угол заливки. Можно также настроить размер области перетекания между сплошными цветами. По окончании создания фонтанной заливки ее можно сохранить для последующего использования в качестве заготовки.

Линейная, радиальная, коническая и прямоугольная фонтанные заливки.

Применение заготовки фонтанной заливки
1.
 
В наборе инструментов выберите инструмент Заливка .
Если требуется выполнить заливку объекта, перед применением заливки выберите объект с помощью инструмента Указатель .
2.
 
Нажмите кнопку Фонтанная заливка на панели свойств.
3.
 
Нажмите кнопку Изменить заливку на панели свойств.
4.
 
В диалоговом окне Фонтанная заливка выберите заготовку фонтанной заливки в списке Заготовки.
5.
 
Нажмите кнопку ОК.
6.
 
Щелкните место на изображении, где необходимо применить заливку.

Дополнительные возможности
 
Указание значения непрозрачности заливки
Введите значение в поле Прозрачность на панели свойств. Чем выше значение, тем больше степень прозрачности.
Указание способа распространения заливки на основе сходства соседних пикселей
Введите значение в поле Допуск на панели свойств. При выборе значения 100 выполняется заливка всего объекта или области.
Изменение способа комбинирования цветов
Выберите режим слияния в списке режимов на панели свойств.

Режимы слияния определяют способ перехода цвета переднего плана или заливки в основной цвет изображения. Можно изменить режим слияния по умолчанию (Среднее) на режим, предназначенный для специальных целей перетекания. Для получения дополнительных сведений о режимах слияния см. раздел Общие сведения о режимах слияния.
Чтобы применить для текстового объекта заливку, можно сначала представить текст в виде редактируемой области, выделив текстовый объект с помощью инструмента Текст и нажав кнопку Создать маску текста на панели свойств. В результате будет создана редактируемая область в форме текста, для которой можно применять заливки.
Создание двухцветной фонтанной заливки
1.
 
В наборе инструментов выберите инструмент Заливка .
2.
 
Нажмите кнопку Фонтанная заливка на панели свойств.
3.
 
Нажмите кнопку Изменить заливку на панели свойств.
4.
 
В диалоговом окне Фонтанная заливка выберите заготовку фонтанной заливки в списке Заготовки.
5.
 
Выберите тип фонтанной заливки в списке Тип.
6.
 
Выберите параметр Двухцветный в области Цветовой переход.
7.
 
Откройте следующие меню выбора цвета и выберите цвет.
•
Из: определение начального цвета последовательного перехода.
•
В: определение конечного цвета последовательного перехода.
8.
 
С помощью регулятора Центр установите среднюю точку между цветами.
9.
 
Выберите один из следующих пунктов.
•
Прямой цветовой путь : обеспечивает перетекание цветов по прямой линии – от начального цвета и (через весь цветовой круг) до конечного.
•
Цветовой путь против часовой стрелки : обеспечивает перетекание цветов по цветовому кругу против часовой стрелки.
•
Цветовой путь по часовой стрелке : обеспечивает перетекание цветов по цветовому кругу по часовой стрелке.

Дополнительные возможности
 
Назначение сдвига центра заливки
Введите значение в поле По горизонтали, По вертикали или в обоих полях. Недоступно для линейных заливок.
Назначение угла линейной или конической заливки
Введите значение в поле Угол.
Назначение количества цветов перехода
Введите значение в поле Шаги. С увеличением значения переход становится более гладким.
Назначение ширины участка, на котором начальный и конечный цвета остаются сплошными до начала перетекания
Введите значение в поле Сдвиг края. Недоступно для конических заливок.
Сохранение заливки в качестве заготовки
Введите имя в списке Заготовки и нажмите кнопку добавления заливки .

Создание специальной фонтанной заливки
1.
 
В наборе инструментов выберите инструмент Заливка .
2.
 
Нажмите кнопку Фонтанная заливка на панели свойств.
3.
 
Нажмите кнопку Изменить заливку на панели свойств.
4.
 
В диалоговом окне Фонтанная заливка выберите заготовку фонтанной заливки в списке Заготовки.
5.
 
Выберите тип фонтанной заливки в списке Тип.
6.
 
Выберите параметр Настройка в области Цветовой переход.
7.
 
Дважды щелкните область над цветовой полосой, чтобы добавить маркер цвета, и щелкните цвет в цветовой палитре.
Если необходимо изменить расположение цветового маркера, перетащите его в новое положение.

Дополнительные возможности
 
Назначение количества цветов перехода
Введите значение в поле Шаги. С увеличением значения переход становится более гладким.
Назначение сдвига центра заливки
Введите значение в поле По горизонтали, По вертикали или в обоих полях.
Назначение угла линейной или конической заливки
Введите значение в поле Угол.
Настройка размера области перетекания между сплошными цветами
Введите значение в поле Сдвиг края. Недоступно для конических заливок.
Установка средней точки между цветами
На цветовой полосе дважды щелкните между двумя цветами, чтобы добавить новый маркер. Перетащите маркер, чтобы установить переход между двумя цветами.
Сохранение заливки в качестве заготовки
Введите имя в списке Заготовки и нажмите кнопку добавления заливки .

Специальные фонтанные заливки могут содержать до 99 цветов.
Маркер цвета можно удалить, дважды щелкнув его.
Чтобы изменить цвет маркера цвета, щелкните маркер цвета, затем щелкните цвет в цветовой палитре.
