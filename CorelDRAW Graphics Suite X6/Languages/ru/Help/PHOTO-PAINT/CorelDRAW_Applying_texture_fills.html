Применение заливок текстурой

Заливки текстурой представляют собой трехмерные узоры. Можно использовать готовые текстурные заливки (вода, минералы, облака), а также создавать собственные на их основе. Файлы, предназначенные для использования в качестве текстурных заливок, не могут быть импортированы.

При редактировании текстурной заливки можно изменять такие параметры, как мягкость, плотность, яркость и цвета. Каждая текстура имеет свои параметры. После редактирования текстурную заливку можно сохранить в качестве пользовательской текстурной заливки.

Для изменения вида текстурной заливки можно изменить ее атрибуты.

Применение заливки текстурой
1.
 
В наборе инструментов выберите инструмент Заливка .
Если требуется выполнить заливку объекта, перед применением заливки выберите объект с помощью инструмента Указатель .
2.
 
Нажмите кнопку Текстура на панели свойств.
3.
 
Нажмите кнопку Изменить заливку на панели свойств.
4.
 
В диалоговом окне Заливка текстурой выберите библиотеку текстур в списке Библиотека текстур.
5.
 
Выберите текстуру в списке Текстура.
6.
 
Нажмите кнопку ОК.
7.
 
Щелкните место на изображении, где необходимо применить заливку.

Дополнительные возможности
 
Редактирование заготовки текстурной заливки
В области Имя стиля диалогового окна Заливка текстурой введите значения в полях параметров текстуры.
Просмотр результатов изменений выбранной текстуры
Нажмите кнопку Просмотр в диалоговом окне Заливка текстурой. При каждом нажатии данной кнопки разблокированные параметры произвольно изменяются, а редактируемая текстура отображается в окне предварительного просмотра.
Сохранение параметров заливки
Нажмите кнопку добавления заливки и введите имя в поле Имя текстуры диалогового окна Сохранение текстуры. Выберите библиотеку в списке Имя библиотеки.
Указание значения непрозрачности заливки
Введите значение в поле Прозрачность на панели свойств. Чем выше значение, тем больше степень прозрачности.
Указание способа распространения заливки на основе сходства соседних пикселей
Введите значение в поле Допуск на панели свойств. При выборе значения 100 выполняется заливка всего объекта или области.
Изменение способа комбинирования цветов
Выберите режим слияния в списке режимов на панели свойств.

Текстурные заливки масштабируются в соответствии с размером изображения или областью изображения, для которой они применяются.
Нельзя создать плитки текстурной заливки.
Чтобы применить для текстового объекта заливку, можно сначала представить текст в виде редактируемой области, выделив текстовый объект с помощью инструмента Текст и нажав кнопку Создать маску текста на панели свойств. В результате будет создана редактируемая область в форме текста, для которой можно применять заливки.
