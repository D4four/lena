GIF

GIF — это формат растровых изображений, созданный специально для использования в Интернете. Данный формат поддерживает изображения, содержащие до 256 цветов, и отличается максимальной степенью сжатия, за счет чего сокращается время передачи файла. Файловый формат GIF поддерживает максимальный размер изображения 30000 30000 пикселей и использует алгоритм сжатия LZW.

Формат GIF позволяет сохранять несколько растровых изображений в одном файле. Файлы GIF с несколькими сменяющими друг друга изображениями называются анимированными файлами GIF. Для получения дополнительных сведений см. раздел Сохранение фильмов.

Изображения GIF с прозрачным фоном часто используются на веб-узлах. Для получения дополнительных сведений см. раздел Создание изображений на основе палитры с прозрачными цветами и фоном.

Для использования в Интернете можно также сохранять изображения в форматах JPEG и PNG. для выбора правильного формата. Если требуется использовать изображение на веб-узле, обратитесь к разделу Выбор веб-совместимого формата файла. для выбора правильного формата.

Импорт файла GIF
1.
 
Выберите Файл Импорт.
С помощью команды Файл Импорт можно расположить файл в виде объекта активного изображения. Если требуется открыть файл GIF в виде изображения, выберите Файл Открыть.
2.
 
Выберите папку, в которой хранится файл.
3.
 
Выполните одно из следующих действий.
•
(Windows 7 и Windows Vista) Выберите вариант GIF - растровое изображение CompuServe (*.gif) в списке рядом с полем Имя файла.
•
(Windows XP) Выберите вариант GIF - растровое изображение CompuServe в списке Тип файла.
4.
 
Щелкните имя файла.
5.
 
Нажмите кнопку Импорт.
6.
 
Щелкните окно изображения.

Открытие анимированного файла GIF
1.
 
Выберите пункт Файл Открыть.
2.
 
Выберите папку, в которой хранится файл.
3.
 
Выполните одно из следующих действий.
•
(Windows 7 и Windows Vista) Выберите вариант GIF — анимация GIF (*.gif) в списке рядом с полем Имя файла.
•
(Windows XP) Выберите GIF – анимация GIF в списке Тип файла.
4.
 
Щелкните имя файла.
5.
 
В списке, расположенном под окном списка, выберите Целиком.
6.
 
Нажмите кнопку Открыть.
Для получения дополнительных сведений об открытии и воспроизведении фильмов см. раздел Открытие и воспроизведение фильмов.
Чтобы открыть отдельный эпизод фильма, выберите пункт Частичная загрузка и в диалоговом окне Частичная загрузка фильма введите значения для диапазона кадров в полях Из и В.
Технические замечания по формату GIF
•
 
Программы Corel импортируют файлы формата GIF версий 87A и 89A, а экспортируют только файлы версии 89A. Версия 87A поддерживает основные функции и чересстрочную развертку. Более новая версия, 89A, поддерживает все функции, присущие версии 87A, а также возможность использования прозрачных цветов и включения комментариев и другой информации из файла изображения.
•
 
При импорте анимированных файлов GIF программы Corel поддерживают изображения со следующей глубиной цвета: черно-белые (1 бит), 16 цветов, оттенки серого (8 бит) и на основе палитры 256 цветов (8 бит).
