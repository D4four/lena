Руководство по программированию макросов

Руководство по программированию макросов CorelDRAW Graphics Suite X6 обеспечивает хорошо организованный учебный подход к программированию макросов для CorelDRAW Graphics Suite X6. Можно использовать среду Microsoft Visual Basic for Applications (VBA) или Microsoft Visual Studio Tools for Applications (VSTA) для создания макросов, автоматизирующих задачи и предоставляющих настраиваемые решения для CorelDRAW и Corel PHOTO-PAINT.
