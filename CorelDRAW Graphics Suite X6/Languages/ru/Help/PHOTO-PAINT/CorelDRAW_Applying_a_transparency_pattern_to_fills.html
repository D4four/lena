Применение узоров прозрачности для заливок

При применении заливки ко всему изображению можно контролировать уровень и узор прозрачности. Заливку можно применить для любого слоя объекта, чтобы создать эффекты переднего плана или фона.

Применение узора прозрачности для заливки
1.
 
Выберите Правка Залить.
2.
 
В диалоговом окне Изменение заливки и прозрачности выберите вкладку Цвет заливки.
3.
 
Выберите один из следующих параметров.
•
Цвет переднего плана
•
Цвет фона
•
Текущая заливка
4.
 
Нажмите кнопку типа заливки.
5.
 
Нажмите Изменить и внесите необходимые изменения в соответствующем диалоговом окне.
6.
 
Перейдите на вкладку Прозрачность.
7.
 
Выберите тип узора в списке Тип.
8.
 
Введите значения в любое из следующих полей.
•
Прозрачность в начале
•
Прозрачность в конце
9.
 
Перетащите курсор в окне отображения, расположенном в верхней части диалогового окна, для установки узлов настройки и просмотрите заливку.
