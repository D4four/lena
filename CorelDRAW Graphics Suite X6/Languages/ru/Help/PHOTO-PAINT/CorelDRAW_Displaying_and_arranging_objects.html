Отображение и упорядочивание объектов

Можно скрыть объект, выровнять его относительно элементов изображения, распределить объекты на изображении, а также изменить порядок размещения объектов.

Отображение и скрытие объектов

По умолчанию в окне изображения отображены все объекты. Однако можно скрыть объект, чтобы сделать его временно невидимым.

Выравнивание объектов

Объекты можно выровнять относительно друг друга, относительно центра изображения, относительно сетки и направляющих или относительно окна изображения. Для получения сведений о выравнивании объектов относительно сетки и направляющих см. раздел Использование направляющих, сетки и линеек.

Распределение объектов

Можно распределить объекты путем задания между ними одинаковых интервалов. Объекты можно распределить по вертикали, по горизонтали или в обоих направлениях. Распределение основывается на указании расстояния между центрами выбранных объектов или на указании интервала между близлежащими краями объектов.

Изменение порядка объектов

При создании на изображении нескольких объектов они накладываются друг на друга в том порядке, в котором создаются. Самый последний созданный объект будет самым верхним, а фон — самым нижним объектом. Можно переместить объект в окне изображения, чтобы закрыть объект, который находится ниже в порядке размещения. Однако объект всегда будет отображаться за объектами, которые находятся выше. При изменении порядка размещения объектов скрытые объекты становятся видимыми или же самые верхние объекты помещаются за другие объекты.

В результате изменения порядка размещения объектов велосипед оказался за коробками.

Скрытие или отображение объекта
•
 
В окне настройки Объекты щелкните значок отображения/скрытия рядом с эскизом объекта.
Если окно настройки Объекты закрыто, выберите пункт Окно Окна настройки Объекты.
Когда объект скрыт, значок отображения/скрытия не отображается.
Когда фон скрыт, отображается сетка прозрачности в клетку. Чтобы настроить отображение прозрачности, выберите Инструменты Параметры. В списке категорий Рабочее пространство выберите пункт Монитор. В диалоговом окне Монитор измените атрибуты в области Таблица прозрачности.
Выравнивание объекта
1.
 
Выделите объект.
2.
 
Выберите Объект Упорядочить Выровнять и распределить.
3.
 
Перейдите на вкладку Выравнивание.
4.
 
Выберите один из следующих параметров.
•
По активному: выравнивание выделенных объектов по активному объекту. Этот параметр доступен, только если выделено несколько объектов.
•
По центру документа: выравнивание выделенного объекта по центру окна изображения.
•
Выбранного для документа: выравнивание выделенного объекта по окну изображения.
5.
 
Установите один из флажков, отвечающих за выравнивание по горизонтали и по вертикали.
Если требуется просмотреть, как будет выглядеть выравнивание в окне изображения, нажмите кнопку предварительного просмотра .
Можно выровнять объект по ближайшей точке сетки, установив флажок Выравнивать по сетке.
Распределение объектов
1.
 
Выделите объекты.
2.
 
Выберите Объект Упорядочить Выровнять и распределить.
3.
 
Перейдите на вкладку Распределение.
4.
 
Выберите один из следующих параметров.
•
По выделению: одинаковые интервалы между выделенными объектами.
•
По документу: одинаковые интервалы между объектами изображения.
•
По интервалу объекта: указание расстояния между объектами в полях X и Y.
5.
 
Установите один из флажков, отвечающих за распределение по горизонтали и по вертикали.
Если требуется просмотреть, как будет выглядеть распределение в окне изображения, нажмите кнопку предварительного просмотра .
Чтобы распределить объекты, требуется выделить несколько объектов.
Изменение порядка размещения объектов
1.
 
Выделите объект.
2.
 
Выберите Объект Упорядочить Порядок, а затем выберите одну из следующих команд.
•
На передний план: размещение выделенного объекта перед остальными объектами изображения.
•
На задний план: размещение выделенного объекта за остальными объектами изображения.
•
На уровень вперед: размещение выделенного объекта перед объектом, за которым он в настоящее время находится.
•
На уровень назад: размещение выделенного объекта за объектом, перед которым он в настоящее время находится.
•
Инвертировать порядок: изменение порядка расположения выделенных объектов на противоположный. Эта команда доступна, только если выделены несколько объектов.
Фон изображения всегда является нижним слоем, под ним нельзя расположить объект.
Когда объекты сгруппированы, считается, что они находятся в порядке размещения на одном уровне. Поэтому невозможно поместить объект между отдельными объектами группы.
Можно изменить порядок размещения объекта, если перетащить его эскиз на новое место в окне настройки Объекты. Если окно настройки Объекты закрыто, выберите пункт Окно Окна настройки Объекты.
Можно также изменить порядок размещения объектов, используя сочетания клавиш. Перемещение объекта на самый вверх — Shift + Page Up; перемещение объекта вниз, но над фоном — Shift + Page Down; перемещение объекта вверх на один уровень — Ctrl + Page Up; перемещение объекта вниз на один уровень — Ctrl + Page Down.
