Создание карт изображений

Карта изображений — это единое графическое изображение с активными областями (или горячими точками), содержащими ссылки на веб-страницы. Горячая точка определяется с помощью координат в изображении, и для каждой определенной области назначается URL-адрес.

Если в изображение требуется добавить ролловеры либо назначить частям изображения разные форматы файлов или степени сжатия, то вместо создания карты изображения его можно разделить на части. Для получения сведений о создании изображений, разделенных на части, см. раздел Разделение изображений.

Создание активных областей

Горячие точки создаются на основе объектов. Для объекта можно назначить URL-адрес и альтернативный текст. Для горячей точки следует также указать форму. Это может быть многоугольник, точно повторяющий форму объекта, прямоугольник, соответствующий рамке выделения объекта, или окружность, в которую вписан объект.

Если требуется создать карту изображения с помощью фотографии, то можно определить редактируемую область, где должна располагаться горячая точка, а затем преобразовать эту область в объект.

Горячая точка на левой кнопке имеет прямоугольную форму, а на правой — округлую. При щелчке в любом месте горячей точки она активируется.

Экспорт карт изображений

При экспорте карты изображения необходимо выбрать один из трех различных типов карт: на клиенте, на сервере или на клиенте и сервере. Тип карты изображения на клиенте используется наиболее широко и является параметром по умолчанию. Далее указаны файлы, которые создаются автоматически в зависимости от того, какой тип карты изображения выбран.

•
 
Страница HTML для следующих типов карт изображений: на клиенте и сервере, на сервере, на клиенте.
•
 
Отдельный файл карты, содержащий координаты горячей точки для следующих типов карт изображений: на клиенте и сервере, на сервере. Для карт изображений на клиенте отдельный файл карты не требуется, так как они содержат теги карты HTML на странице HTML.
Создание активной области для карты изображения
1.
 
В окне настройки Объекты щелкните правой кнопкой мыши эскиз объекта и выберите Свойства.
Если окно настройки Объекты закрыто, выберите пункт Окно Окна настройки Объекты.
2.
 
В диалоговом окне Свойства объекта перейдите на вкладку URL-адрес.
3.
 
Задайте для объекта следующие свойства.
•
URL: адрес (или URL-адрес) веб-страницы, которая будет открываться после щелчка горячей точки. Перед именем домена в веб-адресе необходимо ввести «http://».
•
Примечания: альтернативный текст, который будет отображаться в средстве просмотра при наведении курсора на объект.
•
Определить область: указание формы области горячей точки в объекте.
4.
 
Нажмите кнопку ОК.
Экспорт карты изображения
1.
 
В диалоговом окне Экспортировать для Интернета выберите параметры заготовки в списке Заготовка в верхнем правом углу диалогового окна.
2.
 
В области HTML и части в списке Экспорт выберите Карта изображений.
Если необходимо отобразить карту изображений в средстве просмотра, установите флажок Показать в средстве просмотра.
3.
 
Нажмите кнопку Сохранить как.
4.
 
Выберите диск и папку, в которой необходимо сохранить файл.
5.
 
Введите имя в поле Имя файла.
6.
 
Нажмите кнопку Сохранить.

Дополнительные возможности
 
Ссылка любой части изображения, для которой не назначен URL-адрес, на определенную веб-страницу
В диалоговом окне Сохранение файла карты ссылок установите флажок URL по умолчанию и введите URL-адрес в поле URL по умолчанию.
Добавление информации о файле
Установите флажок Включать заголовок файла.

При экспорте карты изображений объекты будут слиты с фоном.
Кроме того, области горячих точек для карты изображений можно определить с помощью набора инструментов Интернет. Выберите Интернет Объекты Интернета, чтобы отобразить панель инструментов Интернет.
