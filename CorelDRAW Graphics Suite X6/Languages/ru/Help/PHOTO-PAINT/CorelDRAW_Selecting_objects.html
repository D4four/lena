Выделение объектов

Для изменения объектов необходимо их сначала выделить. Можно выделить один объект, закрытые объекты, несколько объектов, все объекты или несколько групп объектов. При выделении одного объекта вокруг него появится поле выделения с восемью маркерами преобразования. При выделении нескольких объектов поле выделения распространяется на все эти объекты.

Можно выделить несколько объектов, однако активным будет только один объект. Активный объект выделен пунктирным контуром, который называется рамкой. Можно изменять активный объект, применяя для него заливку и специальные эффекты.

После внесения изменений в выделенные объекты можно отменить их выделение.

На изображении слева выбран один объект. Справа поле выделения указывает на то, что выделены оба объекта. Вокруг активного объекта отображается рамка выбора объекта.

Выделение объектов

Чтобы выделить
Действие
Объект
В наборе инструментов выберите инструмент Указатель и щелкните объект.
Все объекты изображения
Выберите Объект Выбрать все.
Объект, закрытый другим объектом
В наборе инструментов выберите инструмент Указатель . Удерживая нажатой клавишу Alt, щелкайте, пока поле выделения не захватит отображаемый объект. Эта клавиша позволяет последовательно выделять объекты, расположенные на разных слоях.
Несколько объектов
В наборе инструментов выберите инструмент Указатель . Щелкните объект, удерживая нажатой клавишу Shift, затем щелкните другие объекты.
Несколько групп объектов
В наборе инструментов выберите инструмент Указатель . Щелкните объект одной группы, удерживая нажатой клавишу Shift, затем щелкайте объекты в каждой группе, которую требуется выделить.

Если команда Показывать рамку в меню Объект включена, пунктирный контур, который называется рамкой, будет окружать активный объект.
Если фон выделен до выбора команд Объект Выбрать все, он будет добавлен к выделенным объектам; таким образом будут выделены и объекты, и фон. Если объект был выделен до выбора команды Объект Выбрать все, фон не будет включен в выделенные объекты.
Можно выделить объект, щелкнув его эскиз в окне настройки Объекты. Если окно настройки Объекты закрыто, выберите пункт Окно Окна настройки Объекты.
Можно также выделить объекты в порядке размещения окна настройки Объекты, используя следующие сочетания клавиш: Shift + N — для выбора объекта, который находится поверх текущего объекта; Shift + P — для выбора объекта, который находится под текущим объектом; Shift + T — для выбора самого верхнего объекта и Shift + B — для выбора фонового или самого нижнего объекта.
Отмена выделения объектов

Чтобы отменить выделение
Действие
Объект
В наборе инструментов выберите инструмент Указатель . Щелкните в любом месте за пределами поля выделения объекта.
Несколько объектов
Выберите инструмент Указатель объектов и, удерживая нажатой клавишу Shift, щелкните каждый объект в окне изображения, чтобы отменить их выбор.
Все объекты
Выберите инструмент Указатель и щелкните фон.

При отмене выделения активного объекта он остается активным.
Если фон является частью выделения, то, если щелкнуть фон в окне изображения, отмена выделения всех объектов не удастся.
