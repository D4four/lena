Работа с фильмами QuickTime VR

Corel PHOTO-PAINT позволяет создавать фильмы QuickTime VR. Фильмы QuickTime VR являются виртуальными, трехмерными средами. В этих средах зритель может смотреть вверх, вниз, поворачиваться и выполнять масштабирование.

В фильмах QuickTime VR содержатся точки просмотра, которые называются узлами. Существует два типа узлов QuickTime VR: панорама и объект. Узлы панорамы обеспечивают просмотр изображения на 360° путем вращения вокруг центральной точки просмотра. Широкие изображения лучше всего подходят для панорам. Панораму можно также создать путем плавного соединения или объединения ряда изображений. Для получения дополнительных сведений о соединении изображений см. раздел Объединение изображений.

Узел объекта QuickTime VR является противоположностью узла панорамы: объект закреплен на месте, но его вид меняется при перемещении вокруг него. Узлы объекта QuickTime VR требуют несколько изображений объекта под разными углами и в разных положениях. Изображения объекта не объединяются.

Сцены QuickTime VR представляют собой собрание узлов панорам и объектов, соединенных горячими точками.

Открытие фильмов QuickTime VR

В Corel PHOTO-PAINT можно предварительно просматривать, открывать и редактировать фильмы QuickTime VR. За один раз можно открыть только один узел.

Сохранение и настройка фильмов QuickTime VR

Фильмы с одним кадром (24 бита, RGB) можно сохранять как узлы панорамы QuickTime VR.

Фильмы с несколькими кадрами (24 бита, RGB) можно сохранять как узлы объекта QuickTime VR и настраивать параметры узлов. На объекты QuickTime VR накладывается невидимая сетка. Отображение объекта QuickTime VR зависит от количества строк и последовательности кадров в сетке. При указании количества строк задается количество изображений в каждой строке Например, если в фильме содержится девять кадров, при указании трех изображений для одной строки создаются три строки.

Порядок кадров также является важной деталью, поскольку строки создаются в хронологическом порядке. Например, если в фильме девять кадров в трех строках, они располагаются в следующем порядке: кадры 1–3 в верхней строке, кадры 4–6 в средней строке, кадры 7–9 в нижней строке. Поэтому в кадрах 1–3 объект отображается в перспективе сверху с панорамированием слева направо; кадры 4–6 отображаются в средней перспективе с панорамированием слева направо; кадры 7–9 отображаются в перспективе снизу с панорамированием слева направо. Для получения сведений об изменении порядка кадров фильма см. раздел Изменение последовательности кадров и времени их отображения.

Добавление узла в фильм QuickTime VR

Можно добавлять узлы в фильм или заменять существующие узлы.

Для более быстрой загрузки можно уменьшить разрешение узлов панорамы. Для этого сначала необходимо изменить разрешение изображения. Для получения сведений об изменении разрешения изображений см. раздел Изменение размеров изображения.

Связывание узла QuickTime VR

При сохранении и связывании различных узлов создается среда виртуальной реальности. Объект можно преобразовать в горячую точку, связывающую его с другим узлом или URL-адресом. Чтобы связать объекты, необходимо сохранить текущий узел в существующий фильм QuickTime VR.

Открытие фильма QuickTime VR
1.
 
Выберите пункт Файл Открыть.
2.
 
Выберите пункт MOV — QuickTime VR в списке Тип файла.
3.
 
Выберите папку, в которой хранится файл.
Если необходимо открыть определенный узел фильма с несколькими узлами, перетащите полосу прокрутки Просмотр для выбора сцены, которую требуется открыть.
4.
 
Дважды щелкните имя файла.
Для открытия фильма QuickTime VR на компьютере должна быть установлена программа QuickTime Player 5.0 или более поздней версии.
В Corel PHOTO-PAINT все горячие точки загружаются как объекты.
Сохранение фильма в формате QuickTime VR
1.
 
Выберите пункт Файл Сохранить как.
2.
 
Выберите папку, в которой необходимо сохранить файл.
3.
 
Выберите MOV — QuickTime VR в списке Тип файла.
4.
 
Введите имя файла в поле Имя файла.
5.
 
Нажмите кнопку Сохранить.
6.
 
В диалоговом окне Экспорт QuickTime VR введите имя узла в поле Имя узла.
При сохранении узла объекта укажите количество изображений, которое требуется разместить в одной строке, в поле Число изображений на строку. Количество кадров должно быть кратным количеству строк.
Для использования параметров узлов объекта ширина изображения фильма должна быть кратной четырем пикселям.
В области Параметры элемента управления диалогового окна Экспорт QuickTime VR можно настроить параметры управления узлом объекта с помощью мыши или клавиатуры во время просмотра этого узла в программе QuickTime Player.
Сохранение узла панорамы QuickTime VR
1.
 
Выберите пункт Файл Сохранить как.
2.
 
Выберите папку, в которой необходимо сохранить файл.
3.
 
Выберите MOV — QuickTime VR в списке Тип файла.
4.
 
В поле Имя файла введите имя файла.
5.
 
Нажмите кнопку Сохранить.
6.
 
В диалоговом окне Экспорт QuickTime VR перейдите на вкладку Узел.
7.
 
Введите имя узла в поле Имя узла.
8.
 
Перейдите на вкладку VR World и введите значения в полях Ширина по умолчанию и Длина по умолчанию.
Для сохранения фильма QuickTime VR на компьютере должна быть установлена программа QuickTime Player 5.0 или более поздней версии.
Ширина изображения фильма панорамы должна быть кратной четырем пикселям. Для достижения оптимального результата используйте значение ширины 400 пикселей и значение высоты 300 пикселей.
Перед сохранением файла в качестве узла панорамы QuickTime VR необходимо объединить все объекты с фоном.
Сохранение узла объекта QuickTime VR
1.
 
Выберите пункт Файл Сохранить как.
2.
 
Выберите папку, в которой необходимо сохранить файл.
3.
 
Выберите MOV — QuickTime VR в списке Тип файла.
4.
 
Введите имя файла в поле Имя файла и нажмите кнопку Сохранить.
5.
 
В диалоговом окне Экспорт QuickTime VR перейдите на вкладку Узел.
6.
 
Введите имя узла в поле Имя узла.
7.
 
Укажите количество изображений, которое требуется разместить в одной строке, в поле Число изображений на строку.
8.
 
В области Анимация объекта установите флажок Включить анимацию вида, чтобы фильм воспроизводился сразу после загрузки в программе QuickTime.
Если необходимо, чтобы фильм воспроизводился непрерывно, установите флажок Анимация палиндрома.
9.
 
В области Параметры элемента управления можно настроить параметры управления узлом объекта с помощью мыши или клавиатуры во время просмотра этого узла в программе QuickTime Player путем установки следующих флажков.
•
Перенос: позволяет вращать объект более чем на 360 градусов по горизонтальной оси.
•
Перенос при наклоне: позволяет вращать объект более чем на 360 градусов по вертикальной оси.
•
Включить масштаб: позволяет выполнять масштабирование.
•
Разрешить преобразование объекта: разрешает перемещение объекта при масштабировании.
•
Обращение горизонтального: перемещении мыши по вертикали действует как ее перемещение по горизонтали.
•
Обращение вертикального: перемещении мыши по горизонтали действует как ее перемещение по вертикали.
•
Перемещение вертик. и горизон.: одновременная замена действий, выполняемых при перемещении мыши по вертикали и горизонтали, на обратные.
Для сохранения фильма QuickTime VR на компьютере должна быть установлена программа QuickTime Player 5.0 или более поздней версии.
Добавление узла в фильм QuickTime VR
1.
 
Выберите пункт Файл Сохранить как.
2.
 
Выберите папку, в которую был сохранен фильм QuickTime VR.
3.
 
Выберите MOV — QuickTime VR в списке Тип файла.
4.
 
Дважды щелкните имя файла фильма QuickTime VR.
5.
 
В диалоговом окне Сохранение изображения на диск нажмите кнопку Да.
6.
 
В диалоговом окне Экспорт QuickTime VR введите имя нового узла в поле Имя узла и нажмите кнопку Создать узел.
При сохранении панорамы выберите один из следующих пунктов.
•
Полное: создание узла с исходным разрешением изображения.
•
1/2 от полного: создание узла с разрешением, вдвое меньшим по сравнению с исходным разрешением изображения.
•
1/4 от полного: создание узла с разрешением, вчетверо меньшим по сравнению с исходным разрешением изображения.
•
Просмотр: создание узла с эскизом для просмотра.
Для сохранения фильма QuickTime VR на компьютере должна быть установлена программа QuickTime Player 5.0 или более поздней версии.
Можно перезаписать определенный узел в сцене, выбрав существующий узел в списке.
Связывание узлов QuickTime VR
1.
 
Выберите пункт Файл Сохранить как.
2.
 
Выберите папку, в которой требуется сохранить узел.
3.
 
В поле Имя файла введите имя файла.
4.
 
Выберите MOV — QuickTime VR в списке Тип файла и нажмите кнопку Сохранить.
5.
 
В диалоговом окне Экспорт QuickTime VR перейдите на вкладку Горячие точки.
6.
 
Выберите объект в списке Горячая точка.
7.
 
В области Тип горячей точки выберите один из следующих параметров.
•
Связь: связывание горячей точки с узлом, указанным в списке Связь с.
•
URL: связывание горячей точки с URL-адресом, указанным в поле URL.

Дополнительные возможности
 
Назначение имени для VR World, созданного с помощью связанных узлов
Перейдите на вкладку VR World и введите имя в поле Имя VR World.
Установка одного размера отображения для всех узлов в сцене
Перейдите на вкладку VR World и введите значения ширины и высоты в полях ширины и длины по умолчанию.

Для сохранения фильма QuickTime VR на компьютере должна быть установлена программа QuickTime Player 5.0 или более поздней версии.
В сохраняемом узле должен содержаться объект, чтобы его можно было связать другим узлом или URL-адресом с помощью горячей точки. Для получения сведений о создании объектов см. раздел Создание объектов.
