Обнаружение и встраивание водяных знаков Digimarc

Водяной знак Digimarc на изображении содержит информацию об авторских правах, контактную информацию, а также параметры изображения. В результате внедрения водяных знаков незначительно изменяется яркость пикселей изображения. Эти изменения практически незаметны. Однако после обработки изображения можно заметить изменение яркости некоторых пикселей. Операции редактирования, печати и сканирования никак не влияют на водяные знаки Digimarc.

Обнаружение и встраивание водяных знаков Digimarc не поддерживается 64-разрядной версией CorelDRAW Graphics Suite.

Обнаружение водяных знаков

Открыв изображение в Corel PHOTO-PAINT, можно проверить наличие в нем водяных знаков. Если на изображении имеется водяной знак, в строке заголовка отображается символ авторского права. Информацию об изображении с водяным знаком можно получить, прочитав встроенное сообщение или связавшись с базой данных Digimarc с помощью формы контакта.

Внедрение водяных знаков

С помощью программы Corel PHOTO-PAINT можно также встроить водяные знаки Digimarc в изображения. Сначала необходимо получить уникальный идентификатор создателя, подписавшись на интерактивную службу Digimarc. Идентификатор создателя содержит следующую контактную информацию: имя, номер телефона, почтовый адрес, адреса электронной почты и веб-узла.

Получив идентификатор создателя, можно встроить водяной знак в изображение. Можно также указать год регистрации авторского права, параметры изображения и долговечность водяного знака. На водяном знаке также можно указать заданный метод вывода изображения, например печать или веб-страница.

Водяные знаки Digimarc не обеспечивают защиту от несанкционированного использования изображения и нарушения авторских прав. Однако наличие водяных знаков на изображении указывает на то, что оно защищено авторскими правами. Водяные знаки также содержат контактную информацию для тех, кто заинтересован в получении лицензии или права на использование изображения.

Для получения дополнительных сведений о Digimarc посетите веб-узел www.digimarc.com.

Обнаружение водяного знака
1.
 
Выберите Эффекты Digimarc Прочесть водяной знак.
2.
 
Выберите пункт Веб-поиск для просмотра веб-страницы, содержащей контактную информацию, или позвоните по указанному номеру в службу ответа по факсу Digimarc.
Водяной знак можно также обнаружить при открытии или импорте изображения, установив флажок Проверять наличие водяных знаков.
Данная функция не поддерживается в 64-разрядной версии CorelDRAW Graphics Suite.
Получение идентификатора создателя
1.
 
Выберите Эффекты Digimarc Внедрить водяной знак.
2.
 
Нажмите кнопку Personalize (Личная настройка).
3.
 
В диалоговом окне Personalize creator ID (Личная настройка идентификатора создателя) нажмите кнопку Register (Зарегистрировать) и следуйте инструкциям на веб-узле Digimarc.
4.
 
В диалоговом окне Personalize creator ID (Личная настройка идентификатора создателя) введите идентификационный номер создателя в поле Creator ID (Идентификатор создателя).
Внедрение водяного знака
1.
 
Выберите Эффекты Digimarc Внедрить водяной знак.
2.
 
Заполните поле Copyright year(s) (Год(-ы) регистрации авторских прав) в разделе указания авторских прав.
Нельзя вводить годы ранее 1922 и позднее текущего. При вводе нескольких значений разделяйте их запятой.
3.
 
В области Image attributes (Параметры изображения) установите флажки для используемых параметров.
4.
 
Выберите параметр в списке Target output (Способ вывода).
При выборе параметра Monitor (Монитор) и Web (Интернет) следует установить значение разрешения не менее 200 точек на дюйм, а при выборе параметра Printer (Принтер) — 300 точек на дюйм или выше.
5.
 
Введите значение в поле Watermark durability (Долговечность водяного знака).
Чем выше значение будет установлено, тем больше будет устойчивость водяного знака.
Чтобы подтвердить информацию, которая должна быть доступна для просмотра при обнаружении водяного знака, установите флажок Verify (Проверить).
Данная функция не поддерживается в 64-разрядной версии CorelDRAW Graphics Suite.
Перед добавлением водяного знака необходимо объединить все объекты с фоном. Для получения дополнительных сведений об объединении объектов с фоном см. раздел Группировка и объединение объектов.
