Использование путей для обозначения областей изображения

Инструмент Путь позволяет создавать точные изогнутые линии или очерченные формы на изображении. Пути используются для редактирования маски, ввода текста или нанесения мазков кисти, а также для экспорта изображений неправильной формы, содержащихся в пределах пути.
