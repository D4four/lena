Импорт файлов

В Corel PHOTO-PAINT предусмотрены фильтры, которые при импорте преобразуют файлы из одного формата в другой. Можно импортировать файл и разместить его в активном окне приложения в качестве объекта. Импортированный файл становится частью активного изображения. Перенесенный файл можно также открыть в новом окне приложения.

При импорте файла можно использовать параметры фильтра импорта по умолчанию или выбрать другие необходимые параметры. Встроенный профиль ICC, содержащийся в файле, можно также сохранить в локальную папку Color.

При импорте растрового изображения можно изменить его разрешение, чтобы изменить количество пикселей, исключить неиспользуемые детали и уменьшить размер файла. Растровое изображение можно также обрезать, чтобы точно выбрать область и размер изображения для импорта.

Можно импортировать картинки и фотографии, содержащиеся на DVD-диске Corel. Если используется операционная система Windows 7 или Windows Vista, поиск изображений можно выполнять по разным критериям, например по имени файла, заголовку, теме, автору, ключевому слову, комментарию и другим свойствам файлов. Для получения дополнительных сведений о поиске файлов в ОС Windows 7 или Windows Vista см. справку операционной системы Windows. При использовании операционной системы Windows XP для поиска файлов можно использовать Панель поиска Windows. Кроме того, просмотр и поиск содержимого можно выполнить с использованием Corel CONNECT. Для получения дополнительных сведений см. раздел Знакомство с Corel CONNECT.

Изображения Corel PHOTO-PAINT можно экспортировать в файлы различных форматов. Выбор формата файла зависит от того, как изображение будет использоваться в дальнейшем. Для получения дополнительных сведений об экспорте файлов см. раздел Экспорт изображений в файлы других форматов.

Для получения сведений о конкретных форматах файлов см. раздел Поддерживаемые форматы файлов.

Импорт файла в активное изображение
1.
 
Выберите Файл Импорт.
2.
 
Найдите папку, в которой хранится файл.
3.
 
Выберите формат файла в списке рядом с полем Имя файла (Windows 7 и Windows Vista) или в списке Тип файла (Windows XP).
Если неизвестен формат импортируемого файла, выберите Все форматы файлов.
4.
 
Щелкните имя файла.
Если в файле содержится текст на языке, отличном от языка используемой операционной системы, выберите соответствующий параметр в списке Кодовая страница, чтобы имена, ключевые слова и примечания объектов, сохраненные в файле, отображались правильно. Данный параметр можно применить не ко всем форматам файлов.
5.
 
Нажмите кнопку Импорт.
6.
 
Щелкните окно изображения.

Установите флажок Извлекать встроенный профиль ICC, чтобы сохранить профиль ICC в папку Color, которая находится в папке установки приложения.
Проверка на наличие водяных знаков или сведений об авторских правах
Установите флажок Проверять наличие водяных знаков.
Использование параметров фильтра по умолчанию без открытия диалогового окна
Установите флажок Не показывать диалоговое окно фильтра.
Выберите импортируемую страницу при импорте многостраничных документов TIFF
В диалоговом окне Импорт TIFF выберите параметр в разделе Выберите страницы для импорта.
В приложении Corel PHOTO-PAINT можно открыть только одну страницу многостраничного документа TIFF за один раз.
Поиск изображения (Windows 7 и Windows Vista)
Введите слово или фразу в поле поиска.
С помощью поля поиска выполняется поиск только тех файлов, которые содержатся в текущей папке и подпапках. Чтобы выполнить поиск изображения в другой папке, сначала требуется перейти к папке, в которой хранится искомое изображение.

В диалоговом окне Импорт изображения с диска отображается размер импортируемого изображения и все комментарии, относящиеся к файлу.
Чтобы импортировать файл, можно также открыть его в новом окне приложения. Для получения дополнительных сведений об открытии файла в новом окне приложения см. раздел Открытие изображения.
Изменение разрешения изображения при импорте
1.
 
Выберите Файл Импорт.
2.
 
Выберите папку, в которой хранится файл.
(Windows 7 и Windows Vista) При необходимости можно выполнить поиск изображения с помощью поля поиска. Можно выполнить поиск по таким критериям, как имя файла, заголовок, тема, имя автора, ключевое слово, комментарий, имя растрового изображения, имя объекта и т. д.
3.
 
Выберите формат файла в списке рядом с полем Имя файла (Windows 7 и Windows Vista) или в списке Тип файла (Windows XP).
Если формат файла неизвестен, выберите Все форматы файлов.
4.
 
Щелкните имя файла.
5.
 
Выполните одно из следующих действий.
•
(Windows 7 и Windows Vista) Нажмите кнопку Импорт и выберите Изменить разрешение и загрузить.
•
(Windows XP) Выберите Изменить разрешение в списке рядом с полем Тип файла и нажмите кнопку Импорт.
6.
 
В диалоговом окне Изменение разрешения изображения введите значения в любое из следующих полей.
•
Ширина — ширина рисунка в выбранных единицах измерения или в процентном отношении к исходной ширине.
•
Высота — высота рисунка в выбранных единицах измерения или в процентном отношении к исходной высоте.
7.
 
В области Разрешение введите значения в любое из следующих полей.
•
По горизонтали: разрешение рисунка по горизонтали в пикселях или точках на дюйм (т/д).
•
По вертикали: разрешение рисунка по вертикали в пикселях или точках на дюйм (т/д).
8.
 
Щелкните окно изображения.

Обрезка изображения при импорте
1.
 
Выберите Файл Импорт.
2.
 
Выберите папку, в которой хранится файл.
(Windows 7 и Windows Vista) При необходимости можно выполнить поиск изображения с помощью поля поиска. Можно выполнить поиск по таким критериям, как имя файла, заголовок, тема, имя автора, ключевое слово, комментарий, имя растрового изображения, имя объекта и т. д.
3.
 
Выберите формат файла в списке рядом с полем Имя файла (Windows 7 и Windows Vista) или в списке Тип файла (Windows XP).
Если формат файла неизвестен, выберите Все форматы файлов.
4.
 
Щелкните имя файла.
5.
 
Выполните одно из следующих действий.
•
(Windows 7 и Windows Vista) Нажмите кнопку Импорт и выберите Обрезать и загрузить.
•
(Windows XP) Выберите Обрезка в списке рядом с полем Тип файла и нажмите кнопку Импорт.
6.
 
Введите значения в любое из следующих полей.
•
Верхняя — область, которую требуется удалить в верхней части рисунка.
•
Левая — область, которую требуется удалить в левой части рисунка.
•
Ширина — сохраняемая ширина рисунка.
•
Высота — сохраняемая высота рисунка.
7.
 
Щелкните окно изображения.
При импорте 16-цветные растровые изображения автоматически преобразуются в 256-цветные.
Размер изображения можно также изменить путем перетаскивания поля выделения в окне предварительного просмотра.
Единицы измерения можно изменить, выбрав нужное значение в списке Единицы измерения диалогового окна Обрезка изображения.
