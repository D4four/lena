Работа со сценариями

Сценарии — это короткие программы, в которых для автоматизации простых задач используется язык программирования Corel SCRIPT. Например, если есть несколько недоэкспонированных изображений, можно записать последовательность операций по мере внесения необходимых изменений в первую фотографию. Затем можно воспроизвести эту запись для остальных фотографий и скорректировать их одновременно.

Окно настройки Запись предназначено для создания записей, которые можно сохранять как сценарии для последующего использования. Окно настройки Запись можно также использовать для открытия, редактирования и воспроизведения записей и сценариев.

Для открытия и воспроизведения сценариев можно также использовать окно настройки Сценарии.
Создание записей и сценариев

Можно записать последовательность большинства операций, выполняемых с помощью клавиатуры, панелей инструментов, наборов инструментов, меню и мыши. Записываемые операции преобразовываются в командные операторы, которые появляются в списке команд в хронологическом порядке. Командный оператор — это одно слово, состоящее из имени меню и имени команды из этого меню.

Некоторые операции преобразуются в параметры, которые встраиваются в команду. Параметры записываются, но не отображаются в списке команд. Например, если выбрать цвет краски и нанести на изображение мазок кисти, выбранный цвет не будет отображаться в списке команд, а будет записан в качестве параметра команды инструмента краски.

Следующие операции и команды не могут быть записаны в Corel PHOTO-PAINT:

•
 
настройка панели инструментов, клавиатуры и меню;
•
 
настройка сетки, линейки и направляющих;
•
 
команды оконного меню и меню справки;
•
 
обработка изображений и их объединение;
•
 
операции просмотра, например масштабирование.

Чтобы запись была доступна в следующем сеансе Corel PHOTO-PAINT, ее необходимо сохранить в качестве сценария. Созданные сценарии можно загружать и выполнять в любое время.

В качестве сценария можно также сохранить список отмененных действий. Например, если необходимо воспроизвести эффект, операции создания которого не были записаны, можно сохранить эти операции, используя список отмены. Сценарий, созданный из списка отмены, включает все операции, выполненные при работе с изображением. Чтобы выбрать только необходимые команды, возможно, потребуется отредактировать сценарий.

Редактирование записей и сценариев

Запись или сценарий можно редактировать путем вставки новых команд, перезаписи существующих команд и удаления ненужных команд.

Воспроизведение записей и сценариев

При воспроизведении записи или сценария записанные команды применяются к активному изображению. Запись можно воспроизвести только в текущем сеансе Corel PHOTO-PAINT. Чтобы использовать запись в другом рабочем сеансе, необходимо сохранить ее в качестве сценария. Перед воспроизведением записи или сценария убедитесь, что активное изображение содержит компоненты, необходимые для успешного выполнения записанных команд. Например, если сценарий содержит команды для объектов, его нельзя успешно применить к изображению, не содержащему объектов.

Для изображения можно применить одну команду из записи или сценария. Эта функция удобна, если требуется оценить результат выполнения определенной команды, прежде чем к изображению будут применены остальные команды записи или сценария.

Перед воспроизведением записи или сценария можно временно исключить отдельные команды из последовательности. Для активации отключенных команд не требуется повторно создавать запись или сценарий.

К одному или нескольким изображениям можно одновременно применить либо один, либо несколько сценариев. Такая операция называется пакетной обработкой. Эта функция позволяет вносить глобальные изменения в несколько изображений. При этом нет необходимости открывать каждое изображение и воспроизводить каждый сценарий отдельно. После пакетной обработки изображения можно сохранить в файле исходного или другого формата.

Отображение окна настройки «Запись»
•
 
Выберите Окно Окна настройки Запись.
Отображение окна настройки «Сценарии»
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Рабочее пространство, Настройка выберите вариант Команды.
3.
 
В списке выберите вариант Окно.
4.
 
Перетащите кнопку Сценарии на панель команд и нажмите кнопку OK.
5.
 
Нажмите кнопку Сценарии на панели команд.
Создание записи или сценария
1.
 
Нажмите кнопку Создать в окне настройки Запись.
2.
 
Нажмите кнопку Запись .
3.
 
Выполните действия, которые требуется записать.
4.
 
Нажмите кнопку Остановить .
Запись будет завершена, и ее можно будет воспроизвести в текущем сеансе.
Чтобы сохранить запись в качестве сценария для дальнейшего использования, нажмите кнопку Сохранить , выберите диск и папку, в которой необходимо сохранить сценарий, и введите имя файла в поле Имя файла.
Если первой записанной командой было сохранение документа, можно восстановить исходное изображение, вернувшись к первой команде в записи.
Сохранение списка отмены в качестве сценария
1.
 
Выберите Окно Окна настройки Отменить.
2.
 
Нажмите кнопку Сохранить файл сценария как в окне настройки Отмена.
3.
 
В диалоговом окне Сохранить запись выберите диск и папку, в которой необходимо сохранить файл сценария.
4.
 
В поле Имя файла введите имя файла.
Открытие сценария
1.
 
Нажмите кнопку Открыть в окне настройки Запись.
2.
 
Выберите диск и папку, в которой хранится сценарий.
3.
 
Дважды щелкните имя файла сценария.
Для открытия сценария можно также использовать окно настройки Сценарии.
Вставка команд в запись или сценарий
1.
 
Создайте запись или откройте сценарий в окне настройки Запись.
2.
 
Нажмите кнопку Вставить новую команду .
3.
 
Дважды щелкните команду, которая должна предшествовать вставляемым командам.
Возле выбранной команды отобразится индикатор положения.
4.
 
Нажмите кнопку Запись .
5.
 
Выполните действия, которые требуется добавить.
6.
 
Нажмите кнопку Остановить .
Замена команд в записи или сценарии
1.
 
Создайте запись или откройте сценарий.
2.
 
В окне настройки Запись дважды щелкните первую команду в последовательности, которую требуется заменить.
Возле выбранной команды отобразится индикатор положения.
3.
 
Нажмите кнопку Запись .
4.
 
Выполните новые действия.
5.
 
Нажмите кнопку Остановить .
Удаление команд из записи или сценария
1.
 
Создайте запись или откройте сценарий.
2.
 
В окне настройки Запись щелкните команды, удерживая нажатой клавишу Ctrl.
3.
 
Нажмите кнопку Удалить выбранные команды .
Если из сценария были удалены команды, необходимо сохранить его перед закрытием, чтобы сохранить изменения.
Воспроизведение записи или сценария
1.
 
Создайте запись или откройте сценарий.
2.
 
Нажмите кнопку Воспроизвести в окне настройки Запись.

Дополнительные возможности
 
Воспроизведение одной команды
Дважды щелкните имя команды, которую требуется воспроизвести. (Рядом с выбранной командой отобразится индикатор положения.) Нажмите кнопку Шаг вперед .
Отключение или включение команды
Щелкните команду и нажмите кнопку Включить/отключить выбранные команды .
Имена отключенных команд отобразятся серым цветом.
Переход к первой команде
Нажмите кнопку Перейти в начало .
Переход к последней команде
Нажмите кнопку Перейти в конец .

Для воспроизведения сценария можно также использовать окно настройки Сценарии.
Воспроизведение сценариев для нескольких изображений
1.
 
Выберите Файл Пакетная обработка.
2.
 
Нажмите кнопку Добавить файл.
3.
 
В диалоговом окне Загрузка изображений для запуска макроса выберите диск и папку, в которой хранятся изображения.
4.
 
Удерживая нажатой клавишу Ctrl, щелкните изображения, которые требуется редактировать, и нажмите кнопку Открыть.
5.
 
В диалоговом окне Пакетная обработка нажмите кнопку Добавить сценарий.
6.
 
В диалоговом окне Загрузить сценарий выберите диск и папку, в которой хранятся сценарии.
7.
 
Удерживая нажатой клавишу Ctrl, щелкните сценарии, которые требуется воспроизвести, и нажмите кнопку Открыть.
8.
 
Выберите параметр в списке После выполнения.
Чтобы сохранить файлы в определенную папку, нажмите кнопку Обзор и перейдите к нужной папке.
9.
 
Нажмите кнопку Воспроизвести.
Выберите параметр Не сохранять в списке После выполнения, чтобы просмотреть результаты перед перезаписью исходного изображения.
