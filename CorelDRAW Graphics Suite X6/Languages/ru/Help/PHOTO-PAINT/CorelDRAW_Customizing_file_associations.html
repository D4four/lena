Настройка связей файла

С приложениями Corel можно связать файлы различных типов. Если дважды щелкнуть файл, связанный с приложением, приложение запустится, а файл откроется. Если связь типа файла больше не требуется, ее можно разорвать.

Связь типа файла Corel PHOTO-PAINT
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий дважды щелкните пункт Общие, затем выберите пункт Фильтры.
3.
 
Выберите пункт Связывание.
4.
 
В списке Сопоставлять расширения файлов с Corel PHOTO-PAINT установите флажок рядом с тем типом файла, с которым требуется установить связь.
(Windows 7 и Windows Vista) При связывании типа файла с приложением это приложение будет добавлено в список программ, рекомендуемых для открытия данного типа файла. Чтобы открыть файл, имеющий связь с Corel PHOTO-PAINT в Windows, программа Corel PHOTO-PAINT должна быть задана как программа по умолчанию. Для этого нажмите кнопку Пуск на панели задач Windows, затем выберите пункт Программы по умолчанию. После этого выберите пункт Сопоставление типов файлов или протоколов конкретным программам. Для получения подробных инструкций о том, как изменить программу по умолчанию для типа файла, см. справку Windows.
Можно сбросить параметры связей файла, нажав кнопку Сброс.
Разрыв связи типа файла с Corel PHOTO-PAINT
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий дважды щелкните пункт Общие, затем выберите пункт Фильтры.
3.
 
Выберите пункт Связывание.
4.
 
В списке Сопоставлять расширения файлов с Corel PHOTO-PAINT снимите флажок рядом с тем типом файла, с которым требуется установить связь.
