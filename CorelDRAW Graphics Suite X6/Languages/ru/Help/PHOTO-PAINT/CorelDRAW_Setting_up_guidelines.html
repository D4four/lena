Настройка направляющих

Направляющие представляют собой вертикальные и горизонтальные линии, которые можно добавлять в любом месте в окне изображения и которые служат для измерения, выравнивания и размещения компонентов изображения. Для направляющих используются те же самые единицы измерения, что и для линеек. При сохранении изображения в приложении Corel PHOTO-PAINT направляющие также сохраняются.

Направляющие можно отображать или скрывать. Направляющие можно также добавлять, удалять, перемещать и блокировать в окне изображения.

К направляющим можно привязывать объекты и редактируемые области, чтобы они автоматически выравнивались по направляющей при их перемещении к этой направляющей. Можно настроить чувствительность этой функции, чтобы объект или редактируемая область привязывались при перемещении в пределах определенного расстояния в пикселях от направляющей.

Можно изменять цвет направляющих, чтобы они выделялись на фоне изображения. По умолчанию при выборе направляющей она становится красной. При отмене выбора направляющей она становится синей.

Отображение или скрытие направляющих
•
 
Выберите пункт Вид Направляющие.
Галочка рядом с командой меню указывает на то, что направляющие отображаются.

Направляющие отображаются.

Добавление направляющей
1.
 
Выберите Вид Настройка Направляющие.
2.
 
В списке категорий Направляющие выберите один из следующих пунктов.
•
По горизонтали
•
По вертикали
3.
 
В списке выберите единицу измерения.
4.
 
Введите значение, которое соответствует положению в окне изображения.
5.
 
Нажмите кнопку Добавить.
Направляющую можно также добавить путем перетаскивания из горизонтальной или вертикальной линейки в окне изображения.
Удаление направляющей
1.
 
Выберите Вид Настройка Направляющие.
2.
 
В списке категорий выберите одно из следующих значений.
•
По горизонтали
•
По вертикали
3.
 
В списке выберите направляющую.
4.
 
Нажмите кнопку Удалить.
Все горизонтальные или вертикальные направляющие можно удалить из списка с помощью кнопки Очистить.
Направляющую можно также удалить, перетащив ее за пределы окна изображения.
Перемещение направляющей
1.
 
В наборе инструментов выберите инструмент Выбор объекта .
2.
 
Перетащите направляющую в новое положение в окне изображения.
Направляющую можно также перетащить путем перемещения.
Блокировка или разблокировка направляющих
•
 
Выберите Вид Блокировать направляющие.
Галочка рядом с командой меню указывает на то, что направляющая заблокирована.
Привязка объектов и редактируемых областей к направляющим
•
 
Выберите Вид Привязать к Направляющие.
Настройка чувствительности привязки к направляющим
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий Рабочее пространство выберите пункт Монитор.
3.
 
Введите значение в поле Допуск привязки направляющей (пикселей).
Изменение цвета направляющих
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий Рабочее пространство выберите пункт Монитор.
3.
 
Откройте меню выбора цвета Направляющие и выберите цвет.
