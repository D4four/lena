Идентификация шрифтов

Можно легко определить, какой шрифт используется в тексте веб-страницы или графического дизайна. Приложение Corel PHOTO-PAINT позволяет выполнить захват образца шрифта и автоматически выгрузить его на веб-узел www.whatthefont.com для идентификации.

Идентификация шрифта
1.
 
Выберите Объект Текст WhatTheFont?!
2.
 
Отметьте область со шрифтом, который требуется идентифицировать, с помощью выделяющей рамки.
3.
 
Для выполнения захвата щелкните внутри области захвата или нажмите клавишу Enter.
Для отмены операции нажмите клавишу Esc.
На веб-узле WhatTheFont?! отобразится захваченный шрифт.
4.
 
Следуйте инструкциям, отображаемым на веб-узле WhatTheFont?!, для завершения процедуры идентификации шрифта.
Наиболее подходящая высота букв, при которой обеспечиваются наилучшие результаты поиска, составляет 100 пикселей. Захватывать следует только буквы верхнего или нижнего регистра, но не цифры и не специальные символы. Строки захватываемого текста должны располагаться горизонтально, а буквы не должны соприкасаться друг с другом.
Если требуется выполнить захват текста не из программы Corel PHOTO-PAINT, окно приложения, из которого выполняется захват текста, должно отображаться на экране.
