Вставка связанных или встроенных объектов в другое приложение

Corel PHOTO-PAINT позволяет создавать объекты OLE, которые можно вставлять в другие приложения. Объект можно связать, скопировав его из Corel PHOTO-PAINT и вставив в другое приложение. Можно также встроить объект из Corel PHOTO-PAINT в другое приложение в качестве объекта OLE.

Вставка связанного объекта
1.
 
В Corel PHOTO-PAINT выберите объект.
Убедитесь, что файл сохранен.
2.
 
Выберите пункт Правка Копировать.
3.
 
В конечном приложении выберите пункт Правка Специальная вставка.
4.
 
Выберите параметр Вставить связь.
Вставка встроенного объекта
1.
 
В Corel PHOTO-PAINT выберите объект.
2.
 
Перетащите выбранный объект в окно конечного приложения.
При вставке встроенного объекта путем перетаскивания необходимо, чтобы окна и Corel PHOTO-PAINT, и конечного приложения отображались.
При перетаскивании объекта из одного приложения в другое он удаляется в исходном приложении и копируется в конечное приложение. Если необходимо сохранить копию объекта в Corel PHOTO-PAINT, удерживайте кнопки Ctrl + Shift во время перетаскивания.
