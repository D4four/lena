Открытие изображений

В Corel PHOTO-PAINT можно открывать большинство растровых изображений. Каждое открытое изображение отображается в отдельном окне.

Кроме того, изображения можно импортировать. С помощью команды импорта можно добавить новое изображение в окно активного изображения. Для получения дополнительных сведений см. раздел Импорт файлов.

Можно использовать картинки и фотографии, содержащиеся на DVD-диске Corel. Если используется операционная система Windows 7 или Windows Vista, поиск изображений можно выполнять по разным критериям, например по имени файла, заголовку, теме, автору, ключевому слову, комментарию и другим свойствам файлов. Для получения дополнительных сведений о поиске файлов в ОС Windows 7 или Windows Vista см. справку операционной системы Windows. При использовании операционной системы Windows XP для поиска файлов можно использовать Панель поиска Windows. Кроме того, просмотр и поиск содержимого можно выполнить с использованием Corel CONNECT. Для получения дополнительных сведений см. раздел Знакомство с Corel CONNECT.

Открытие более ранних версий многоязычных файлов

Изображение, содержащее текст на языке, отличном от языка используемой операционной системы, можно также открыть или импортировать из версии 11 или более ранней версии Corel PHOTO-PAINT. Для этого с помощью параметров кодовой страницы необходимо настроить правильное отображение сохраненных вместе с изображением имен и примечаний в окне настройки Объекты. Для правильного отображения текста в окне изображения необходимо настроить параметры кодировки. Для получения дополнительных сведений см. раздел Изменение параметров кодировки для правильного отображения текста.

Открытие изображения
1.
 
Выберите пункт Файл Открыть.
2.
 
Найдите папку, в которой хранится файл.
3.
 
Щелкните имя файла.
При необходимости можно выполнить поиск изображения с помощью поля поиска. Можно выполнить поиск по имени файла, заголовку, теме, автору, ключевому слову или комментарию.
Для выполнения поиска определенных пользователем меток в Windows XP на компьютере должна быть установлена Панель поиска Windows.
4.
 
Нажмите кнопку Открыть.

Дополнительные возможности
 
Обнаружение водяного знака
Установите флажок Проверять наличие водяных знаков.
Данный параметр можно применить не ко всем форматам файлов.
Удаление ненужных областей изображения при его открытии
(Windows 7 и Windows Vista) В списке Открыть выберите Обрезать и загрузить.
(Windows XP) В списке Тип файла выберите Обрезка.
Уменьшение размеров изображения при его открытии
(Windows 7 и Windows Vista) В списке Открыть выберите Изменить разрешение и загрузить.
(Windows XP) В списке Тип файла выберите Изменить разрешение.
Поиск изображения (Windows 7 и Windows Vista)
Введите слово или фразу в поле поиска.
С помощью поля поиска выполняется поиск только тех файлов, которые содержатся в текущей папке и подпапках. Чтобы выполнить поиск изображения в другой папке, сначала требуется перейти к папке, в которой хранится искомое изображение.
Открытие предыдущей версии файла (Windows 7 и Windows Vista)
Щелкните правой кнопкой мыши файл и выберите Восстановить предыдущие версии.
Предыдущую версию файла можно открыть, только если включена защита системы.
Для получения дополнительных сведений об открытии предыдущих версий файлов см. справку Windows.
Правильное отображение имен объектов и примечаний в изображении, созданном в более ранней версии Corel PHOTO-PAINT
(Windows 7 и Windows Vista) Выберите соответствующий параметр в списке Выбрать кодовую страницу.
(Windows XP) Выберите соответствующий параметр в списке Кодовая страница.
Данный параметр можно применить не ко всем форматам файлов.

Изображение можно также открыть, нажав кнопку Открыть на стандартной панели инструментов. Если стандартная панель инструментов не отображается, выберите Окно Панели Стандартная.
Вставка файла в активное изображение
•
 
Перетащите файл изображения или звуковой файл в окно изображения.
Чтобы открыть файл в новом окне изображения, можно также перетащить его в окно приложения.
