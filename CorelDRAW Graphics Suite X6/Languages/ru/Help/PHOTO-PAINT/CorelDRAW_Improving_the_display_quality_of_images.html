Повышение качества отображения изображений

Если используется монитор, отображающий менее 16 миллионов цветов (24-разрядный цвет), то для улучшения качества отображения изображений можно использовать смешивание. Во время смешивания пиксели с определенными значениями цветов размещаются относительно других пикселей. Взаимосвязь одного цветного пикселя с другим приводит к появлению дополнительных цветов, которые отсутствуют в цветовой палитре.

Повышение качества отображения изображения
•
 
Выберите Вид Смешивание, а затем выберите один из следующих параметров.
•
Нет: отключение функции смешивания, когда компьютер находится в 16-разрядном цветовом режиме.
•
Диффузия: распространение смешивания на более широкую область и использование узора смешивания для имитируемых переходов.
•
Упорядоченное: сближение цветовых переходов с помощью фиксированной точечной структуры. Этот тип смешивания работает быстрее, чем тип Диффузия, однако при этом получается менее точный результат.
