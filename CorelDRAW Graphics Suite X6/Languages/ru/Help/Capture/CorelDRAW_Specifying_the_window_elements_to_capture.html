Выбор элементов окна для захвата

Можно выбрать окно для захвата, например активное диалоговое окно или его часть. Кроме того, для захвата можно выбрать меню.

Захват областей, определенных пользователем, не выполняется автоматически после нажатия сочетания клавиш. Для этого необходимо указать ту область изображения, захват которой требуется выполнить. Например, можно выделить рамкой прямоугольную, эллиптическую область изображения или область свободной формы. При захвате эллиптической области или области свободной формы можно изменить цвет фона изображения.

Пользовательские настройки можно сохранить для дальнейшего использования. Например, можно один раз указать параметры для изображения, которое будет использоваться на веб-узле, и сохранить их. Эти пользовательские параметры можно удалить, если они больше не требуются.

Выбор окна для захвата
1.
 
Перейдите на вкладку Объект.
2.
 
Выберите параметр Захват окна.
3.
 
В списке Захват окна выберите один из следующих пунктов.
•
Текущее окно: захват активного окна.
•
Окно клиента: захват активного окна, кроме строки заголовка, строки состояния и рамки окна.
•
Анимация: позволяет остановить захват, нажав сочетание клавиш, и сохранить все, что было захвачено до этого момента.
•
Во весь экран: захват всей области рабочего стола.
•
Текущий объект: позволяет выполнить захват части активного окна, например кнопки или списка.
•
Текущий объект с рамкой: позволяет заключить необходимый для захвата объект в рамку.
•
Окно с прокруткой: позволяет прокрутить окно и выполнит захват его содержимого.
Выбор меню для захвата
1.
 
Перейдите на вкладку Объект.
2.
 
Выберите параметр Захват меню.
3.
 
Нажмите кнопку Настройка.
4.
 
В списке Настройка выберите один из следующих пунктов.
•
Каскадное меню: захват каскадного меню.
•
Включить меню: включение строки меню при захвате.
•
Заменить фон: позволяет заменять фон позади меню путем его закрытия цветом, выбранным в списке Цвет фона.
Выбор пользовательской области для захвата
1.
 
Перейдите на вкладку Объект.
2.
 
Выберите параметр Захват области.
3.
 
В списке Захват области выберите один из следующих пунктов.
•
Прямоугольная область: позволяет выбрать прямоугольную область изображения для захвата.
•
Эллиптическая область: позволяет выбрать эллиптическую область изображения для захвата.
•
Область свободной формы: позволяет определять область изображения для захвата.
При захвате области свободной формы можно указать для нее до 50 углов.
Выбор цвета фона
1.
 
Перейдите на вкладку Объект.
2.
 
Выберите параметр Захват области.
3.
 
В цветовой палитре Цвет фона выберите цвет.
Цвет можно менять только при захвате областей эллиптической и свободной формы.
Сохранение пользовательских параметров
1.
 
Перейдите на вкладку Объект.
2.
 
Щелкните Плюс.
3.
 
В поле Имя файла введите имя файла.
Удаление пользовательских параметров
1.
 
Перейдите на вкладку Объект.
2.
 
Выберите пользовательские параметры в списке Заготовки захвата.
3.
 
Щелкните Минус.
