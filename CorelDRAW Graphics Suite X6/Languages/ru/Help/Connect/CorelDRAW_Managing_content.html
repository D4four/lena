Управление содержимым

Содержимое можно систематизировать с помощью лотков или путем добавления файлов в определенные папки на компьютере.

В лотке можно собирать содержимое из разных папок. К файлам в лотке можно обращаться, но в действительности они остаются в своем исходном местоположении. Содержимое можно добавлять в лоток и удалять из него. Лоток является общим для CorelDRAW, Corel PHOTO-PAINT и Corel CONNECT. Можно также открывать файлы из лотка, что упрощает их предварительный просмотр.

Лотки позволяют систематизировать содержимое. Лотки хранятся в пользовательской папке в файлах с расширением .Tray. Можно изменить папку по умолчанию, в которой хранятся лотки. Например, можно сохранить лоток в сетевой папке, чтобы использовать его совместно с другими пользователями.

Лотки можно переименовывать, чтобы их имена были содержательными. Во время работы с несколькими лотками можно временно закрыть некоторые лотки, чтобы избежать беспорядка. Лотки можно легко открыть повторно, когда они потребуются. Кроме того, можно загрузить файл лотка, который отсутствует в списке доступных лотков. Лоток можно удалить в любое время.

Добавление содержимого в лоток
•
 
Выберите один или несколько эскизов в области просмотра и перетащите их в лоток.
Если необходимо открыть файл в лотке, дважды щелкните его эскиз.
Кроме того, можно перетащить содержимое из папки на компьютере в лоток.
Добавление содержимого в папку
•
 
Выберите один или несколько эскизов в области просмотра и перетащите их в папку области Избранные папки, Папки или в любую другую папку на компьютере.
Содержимое можно перетащить из лотка в папку.
Удаление содержимого из лотка
•
 
Выберите один или несколько эскизов в лотке и нажмите кнопку Удалить из лотка .
Создание лотка
•
 
Нажмите кнопку Добавить лоток , расположенную в правом нижнем углу панели лотка.
По умолчанию лотки сохраняются в папке Users\[UserName]\Documents\Corel\Corel Content\Trays.
Лотки, содержащие только интернет-контент, можно отправить по электронной почте.
Изменение папки, в которую сохраняются лотки
1.
 
Щелкните стрелку рядом с кнопкой Справка и параметры и выберите Параметры.
2.
 
Введите путь в поле Стандартное расположение лотка.
Чтобы просмотреть папку, нажмите кнопку Обзор рядом с полем Стандартное расположение лотка.
Для совместного использования лотка его можно сохранить в сетевую папку.
Работа с лотками

Цель
Действие
Выбор лотка
Перейдите на вкладку лотка.
Переименование лотка
Щелкните правой кнопкой мыши вкладку лотка, выберите пункт Переименовать и задайте новое имя лотка.
Открытие лотка
Щелкните стрелку Все лотки и выберите лоток.
Рядом с именем открытого лотка отображается флажок.
Закрытие лотка
Щелкните правой кнопкой мыши вкладку лотка и выберите Закрыть.
Загрузка файла лотка
Щелкните стрелку Все лотки, выберите Открыть файл лотка и перейдите к файлу лотка.
Перемещение между несколькими лотками
Удерживая нажатой клавишу Ctrl, нажмите клавишу Tab.
Удаление лотка
Щелкните правой кнопкой мыши вкладку лотка и выберите команду Удалить.

Можно также закрыть и удалить выбранный лоток с помощью меню Все лотки.
