Экспорт в формате HTML

Чтобы убедиться, что файлы и объекты CorelDRAW успешно опубликованы в формате HTML, настройте элементы документа так, чтобы они были HTML-совместимыми, выберите нужные настройки и проверьте все необходимое при подготовке. Затем их можно опубликовать в формате HTML. Получаемый в результате код HTML и изображения можно использовать в программе редактирования файлов HTML для создания веб-узла или страницы.

В этом разделе рассматриваются следующие темы.

•
 
Установка параметров экспорта изображений в формате HTML
•
 
Создание веб-совместимого текста
•
 
Предварительный просмотр и проверка файлов перед экспортом в формате HTML
