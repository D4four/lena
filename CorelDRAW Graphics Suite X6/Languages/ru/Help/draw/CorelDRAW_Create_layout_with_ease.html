Создавайте макеты с легкостью

Благодаря новым и улучшенным функциональным возможностям слоя-шаблона, новым временным направляющим выравнивания, новой расширенной поддержке OpenType и расширенной поддержке сложных сценариев для работы с текстом на иностранных языках, работать над проектами в CorelDRAW Graphics Suite X6 стало еще проще.

Слои-шаблоны (новые и улучшенные!)

В CorelDRAW X6 новые и улучшенные слои-шаблоны для нечетных, четных и всех страниц упрощают разработку дизайна страниц для многостраничных документов. При разработке макетов многостраничных документов, таких как брошюры и листовки, в них теперь можно легко добавить колонтитулы и номера страниц. Например, может потребоваться расположить на четных страницах один тип заголовка, а на нечетных страницах — другой тип заголовка.

Нумерация страниц (новинка!)

В CorelDRAW X6 новая команда «Вставить номер страницы» помогает мгновенно добавить номера страниц на всех страницах документа, начиная с определенной страницы или определенного номера. Такая гибкость идеально подходит для работы с несколькими файлами CorelDRAW, которые в конечном итоге станут одной публикацией.

Можно выбрать буквенную, числовую или римскую нумерацию, а также задать для буквенной нумерации строчное или прописное начертание. Кроме того, номера страниц автоматически обновляются при добавлении или удалении страниц в документе. Номера страниц можно также вставить внутри существующего фигурного или обычного текста.

Направляющие выравнивания (новинка!)

В CorelDRAW X6 новые направляющие выравнивания помогают быстрее расположить объекты благодаря контекстному отображению для существующих объектов на странице. Эти временные направляющие появляются при создании объектов, изменении их размера и перемещении относительно других соседних объектов. Направляющие интерактивно привязываются центрам и краям объектов. Можно также включить отображение направляющих от краев одного объекта к центру другого объекта.

Параметры по умолчанию для направляющих можно настроить согласно своим потребностям. Например, при работе с группой объектов можно отобразить направляющие для отдельных объектов в группе или для границы группы в качестве единого целого. Кроме того, для направляющих выравнивания можно указать отступы, чтобы помочь выровнять объекты на заданном расстоянии. Можно также отобразить направляющие выравнивания таким образом, чтобы привязка выполнялась только с учетом размера отступа или на размер отступа с учетом фактических краев объекта в дополнение к отступам.

Интерактивные фреймы (новинка!)

В CorelDRAW X6 доступны интерактивные фреймы, которые помогают эффективно создавать макеты для дизайнерских идей. Функциональность новых пустых контейнеров PowerClip и текстовых фреймов позволяет использовать в проектах местозаполнители контейнеров PowerClip и текстовых фреймов, что упрощает предварительный просмотр макета перед завершением его отдельных компонентов содержимого.

В CorelDRAW X6 теперь можно перетащить содержимое на контейнер PowerClip, а затем добавить содержимое во фрейм или заменить какое-либо имеющееся содержимое фрейма. Фреймы PowerClip теперь также позволяют центрировать или пропорционально масштабировать содержимое в пределах фрейма. Кроме того, теперь можно легко создавать текстовые фреймы на основе любого закрытого объекта кривой, а затем редактировать фрейм для реализации любой формы, которую только можно вообразить.

Функциональные возможности интерактивных фреймов также были усовершенствованы в коллекции готовых графических фреймов, входящих в состав пакета, предоставляя вам готовые стильные дизайнерские материалы, которые можно легко настроить.

Шаблонный текст (новинка!)

В CorelDRAW X6 новая команда «Вставить шаблонный текст» позволяет щелкнуть правой кнопкой мыши любой текстовый фрейм и мгновенно заполнить его шаблонным текстом. Это упрощает оценку внешнего вида документа перед добавлением фактического содержимого.

Для повышения удобства и гибкости можно также использовать свой шаблонный текст на любом языке, который поддерживается в CorelDRAW. Просто создайте RTF файл, содержащий необходимый текст, сохраните его в папке шаблонного текста по умолчанию и CorelDRAW X6 будет использовать этот текст при каждом вызове команды «Вставить шаблонный текст». Дополнительные сведения о создании пользовательского файла шаблонного текста см. в разделе Вставка шаблонного текста.

Для документов можно создавать свой шаблонный текст.

Расширенная поддержка OpenType (новинка!)

В CorelDRAW Graphics Suite X6 обновленный текстовый модуль позволяет воспользоваться дополнительными преимуществами возможностей типографики OpenType, такими как контекстные и стилистические варианты начертания, дроби, лигатуры, порядковые числительные, орнаменты, капители, росчерки и многое другое. Шрифты OpenType основаны на Юникоде, что делает их идеальным выбором для кросс-платформенных проектов. Кроме того, расширенные наборы символов обеспечивают превосходную языковую поддержку.

Шрифты OpenType позволяют выбрать альтернативный вид для отдельных символов.

Возможности OpenType доступны в окне настройки «Свойства объекта» и позволяют выбирать альтернативные начертания для отдельных символов или глифов в соответствии с вашими стилистическими предпочтениями при условии, что шрифт поддерживает дополнительную функциональность OpenType. Например, можно задействовать другой глиф числа, дроби или лигатуры, чтобы изменить вид своего текста. Кроме того, с помощью функции «Интерактивный OpenType» приложение CorelDRAW X6 обеспечит подходящие возможности OpenType, доступные для применения к тексту.

Поддержка сложных сценариев (улучшенная!)

В CorelDRAW X6 благодаря поддержке сложных сценариев, встроенной в переработанную систему обработки текста, обеспечивается надлежащий ввод текста для глифов, используемых в азиатских и ближневосточных языках.

Улучшения системы обработки текста обеспечивают правильное отображение многоязычных глифов в документах.
