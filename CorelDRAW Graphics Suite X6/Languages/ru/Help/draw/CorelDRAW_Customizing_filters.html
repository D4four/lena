Настройка фильтров

Фильтры используются для преобразования файлов из одного формата в другой. Они бывают четырех типов: растровые, векторные, анимационные и текстовые. Можно настроить параметры фильтра путем добавления или удаления фильтров, чтобы загружать только необходимые фильтры. Кроме того, можно изменить порядок расположения фильтров в списке, а также сбросить фильтры до параметров по умолчанию.

Добавление фильтра
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий дважды щелкните пункт Общие, затем выберите пункт Фильтры.
3.
 
Дважды щелкните тип фильтра в списке Доступные типы файлов.
4.
 
Щелкните фильтр.
5.
 
Нажмите кнопку Добавить.
Удаление фильтра
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий дважды щелкните пункт Общие, затем выберите пункт Фильтры.
3.
 
Щелкните фильтр в списке Список активных фильтров.
4.
 
Нажмите кнопку Удалить.
Изменение порядка расположения фильтров в списке
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий дважды щелкните пункт Общие, затем выберите пункт Фильтры.
3.
 
Щелкните фильтр в списке Список активных фильтров.
4.
 
Выберите один из следующих пунктов.
•
Вверх — перемещение фильтра по списку вверх.
•
Вниз — перемещение фильтра по списку вниз.
Можно сбросить Список активных фильтров до параметров по умолчанию, нажав кнопку Сброс.
