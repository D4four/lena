Использование Bitstream Font Navigator

Bitstream Font Navigator — это система управления шрифтами, включенная в приложение. Bitstream Font Navigator позволяет хранить до 2000 шрифтов в базе данных (или каталоге). Можно получить быстрый доступ к шрифтам, организовать их, а также просмотреть до использования. Для получения дополнительных сведений см. раздел справки в Bitstream Font Navigator.

Запуск Bitstream Font Navigator
•
 
На панели задач Windows выберите Пуск Все программы CorelDRAW Graphics Suite X6 Bitstream Font Navigator.
Открытие справки Bitstream Font Navigator
•
 
В Bitstream Font Navigator выберите пункт Справка Вызов справки.
