Получение изображений со сканеров и цифровых камер

В CorelDRAW можно загружать фотографии с цифровых камер и сканировать изображения.

32-разрядная версия CorelDRAW поддерживает сканеры, использующие стандартный интерфейс Microsoft Windows Image Acquisition (WIA) для загрузки изображений. Если сканер или цифровая камера не поддерживает интерфейс WIA, для загрузки изображений можно использовать драйверTWAIN сканера или цифровой камеры. TWAIN поддерживается 32- и 64-разрядными версиями CorelDRAW. Интерфейсы и параметры в разных версиях программного обеспечения могут различаться. Для получения сведений об использовании программного обеспечения, имеющего отношение к используемому сканеру или цифровой камере, см. документацию производителя.

Загрузка фотографий с цифровой камеры
1.
 
Подключите цифровую камеру к компьютеру.
2.
 
Выберите Файл Получить изображение Выбрать источник.
3.
 
В поле Источники выберите цифровую камеру.
Цифровая камера может иметь оба драйвера: WIA и TWAIN.
4.
 
Выберите Файл Получить изображение Получить.
5.
 
В появившемся диалоговом окне выберите изображения, которые требуется загрузить.
Если цифровая камера не поддерживает интерфейс WIA, для загрузки изображений будет предложено использовать интерфейс драйвера TWAIN цифровой камеры. Параметры разных цифровых камер могут различаться.
6.
 
Нажмите кнопку Получить изображения.
Название этой кнопки в интерфейсе цифровой камеры может быть другим.
Для загрузки других фотографий во время того же сеанса выберите Файл Получить изображение Получить.
Если цифровая камера не поддерживает интерфейс WIA или не имеет драйвера TWAIN, в CorelDRAW все равно можно открывать фотографии, для чего необходимо выбрать Файл Импорт, найти требуемую папку в памяти цифровой камеры и выбрать фотографии, которые необходимо открыть.
Сканирование изображений
1.
 
Выберите Файл Получить изображение Выбрать источник.
2.
 
Выберите сканер в списке Источники.
У сканера может быть два драйвера: WIA и TWAIN. При сканировании 48-битовых цветных изображений необходимо выбрать драйвер TWAIN.
3.
 
Нажмите кнопку Выбрать.
4.
 
Выберите Файл Получить изображение Получить.
Если сканер не поддерживает интерфейс WIA, для загрузки изображений будет предложено использовать интерфейс драйвера TWAIN сканера. Параметры в разных сканерах могут различаться.
5.
 
В окне предварительного просмотра изображения выберите область, которую необходимо сканировать.
6.
 
Нажмите кнопку Сканирование.
Название этой кнопки в интерфейсе сканера может быть другим. Она может называться ОК или Отправить.
Интерфейс WIA доступен только в операционной системе Windows XP.
Для сканирования других изображений во время того же сеанса выберите Файл Получить изображение Получить.
