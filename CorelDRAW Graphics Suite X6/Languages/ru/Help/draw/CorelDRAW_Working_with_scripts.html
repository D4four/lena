Работа со сценариями

Сценарии — это короткие программы, в которых для автоматизации простых задач используется язык программирования Corel SCRIPT. Например, можно воспроизвести сценарий, который применяет заготовку заливки к выбранному объекту.

При использовании CorelDRAW версий с 6 по 9 единственным способом автоматизации задач было использование языка Corel SCRIPT для создания сценариев. Модуль Visual Basic for Applications (VBA), который поддерживается приложением CorelDRAW начиная с версии 10, можно использовать для создания макросов, которые автоматизируют более сложные задачи, чем сценарии Corel SCRIPT. Несмотря на то что CorelDRAW по- прежнему поддерживает язык Corel SCRIPT, для автоматизации задач настоятельно рекомендуется использовать макросы, а не сценарии. Для получения дополнительных сведений см. раздел Работа с макросами.

Для отображения и воспроизведения сценариев можно использовать окно настройки Диспетчер сценариев Corel в окне приложения. Чтобы воспроизвести сценарии, можно также воспользоваться меню Инструменты.

Отображение окна настройки диспетчера сценариев Corel
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Рабочее пространство, Настройка выберите вариант Команды.
3.
 
В списке выберите вариант Инструменты.
4.
 
Перетащите кнопку Диспетчер сценариев Corel на панель команд и нажмите кнопку OK.
5.
 
Нажмите кнопку Диспетчер сценариев Corel на панели команд.
Воспроизведение сценария
1.
 
Выберите Инструменты Запустить сценарий.
2.
 
Выберите сценарий, который требуется воспроизвести.
3.
 
Нажмите кнопку Открыть.
Сценарий можно также воспроизвести в окне настройки Диспетчер сценариев Corel.
