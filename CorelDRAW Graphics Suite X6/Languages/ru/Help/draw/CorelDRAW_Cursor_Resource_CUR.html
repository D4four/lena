Ресурс курсоров (CUR)

Формат файлов ресурса курсоров Windows 3.x/NT (файлы .cur) используется для создания курсоров интерфейсов Windows 3.1, Windows NT и Windows 95. Он поддерживает графические элементы, используемые в указателях Windows. Можно выбрать цвет прозрачных и инвертированных масок.

Максимальный поддерживаемый размер изображения в формате файла ресурсов курсоров Windows 3.x/NT – 32 32 пикселя.

При импорте файлов .cur программы Corel поддерживают изображения со следующей глубиной цвета: черно-белые (1 бит), на основе палитры 16 цветов (4 бита) и на основе палитры 256 цветов (8 бит).
