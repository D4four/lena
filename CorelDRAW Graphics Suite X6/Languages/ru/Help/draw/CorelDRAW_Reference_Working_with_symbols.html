Изменение экземпляров символа

Для экземпляров символа можно изменить следующие свойства объекта. Если символ содержит несколько объектов, то все объекты в экземпляре символа определяются как один объект, как если бы они принадлежали к одной группе.

Свойство
Заметки
Положение
Можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Расположение объектов.
Размер
Можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Изменение размера и масштабирование объектов.
Масштаб
Значение в процентах, зависящее от определения символа. Можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Изменение размера и масштабирование объектов.
Наклон
Для получения дополнительных сведений см. раздел Наклон и растягивание объектов.
Угол поворота
Можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Поворот и зеркальное отражение объектов.
Прозрачность
Только однородная прозрачность. Прозрачность можно изменить в разделе Символ окна настройки Свойства объекта.
Отражение
Можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Поворот и зеркальное отражение объектов.
Перенос текста по словам
Режим переноса по словам можно изменить в разделе Сводка окна настройки Свойства объекта. Также можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Выполнение обтекания текста вокруг объекта, фигурного текста или рамки.
Порядок
Можно изменить с помощью элементов панели свойств. Для получения дополнительных сведений см. раздел Изменение порядка объектов.
Имя
Имя экземпляра отображается в окне настройки Диспетчер символов. Дважды щелкните, чтобы изменить его.

Неподдерживаемые типы объектов

Большинство объектов CorelDRAW, за исключением следующих объектов, можно преобразовать в символы.

Неподдерживаемый тип объекта
Заметки
Связанные или встроенные объекты
Включены штрихкоды
Объекты с линзами, которые не являются застывшими
Линзы должны быть застывшими.
Объекты привязки в группах связи (падающие тени, контуры, перетекания, скосы и вытягивания)
Все связанные объекты в группе связи должны быть включены в символ. Например, если объект имеет падающую тень, его невозможно преобразовать без тени. В качестве другого способа можно разъединить группу.
Объекты, к которым применены художественные эффекты
Эффекты распыления должны быть разъединены в объекты кривых. Другие эффекты художественного оформления должны быть разделены на части или преобразованы как группа связи. Эффекты распыления, содержащие прозрачности, не могут быть преобразованы в символы.
Поля слияния при печати
 
Связанные растровые изображения
 
Размещенные файлы PDF или EPS
 
Заблокированный объект
Объект должен быть разблокирован.
Простой текст
 
Соединительные и размерные линии
Должны быть разъединены или включены со связанным объектом или объектами
Направляющие
 
Ролловеры
Невозможно преобразовать ролловер в символ. Однако можно включить экземпляр символа в состояние ролловера.
Объекты, для которых применена прозрачность или эффекты прозрачности
Эффекты прозрачности должны быть преобразованы в растровое изображение.
