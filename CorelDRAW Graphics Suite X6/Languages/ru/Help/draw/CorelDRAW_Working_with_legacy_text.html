Работа с текстом из прежних версий

Если открыть документ с текстом, который был создан в предыдущей версии CorelDRAW, например в CorelDRAW X5, старый текст необходимо обновить, прежде чем для него можно будет использовать новые функции работы с текстом CorelDRAW Х6, такие как OpenType и базовая сетка. Для получения дополнительных сведений см. разделы Работа с возможностями OpenType и Выравнивание текста с помощью базовой сетки.

Обновление текста из прежних версий
1.
 
Откройте документ, содержащий текст из предыдущих версий приложения.
Панель инструментов Обновление отображается в верхней части окна рисования.
2.
 
Нажмите кнопку Обновить.
Чтобы отменить обновление, щелкните Отменить на панели инструментов Обновление. Однако после внесения изменений в обновленный текст этот параметр становится недоступным.
Можно также обновить унаследованный текст путем нажатия кнопки Обновить в области Символ окна настройки Свойства объекта.
