Установка параметров экспорта изображений в формате HTML

Можно установить параметры экспорта изображений, предназначенных для просмотра в Интернете. Например, можно определить параметры макета страницы, такие как размер и расположение пробела изображения.

Кроме того, документ можно опубликовать в виде отдельного изображения, из которого приложение создает карту изображений. Карта изображений – это гиперграфика, горячие точки которой при просмотре документа в формате HTML с помощью средства просмотра являются ссылками на различные URL-адреса, включая страницы, папки и изображения. Следует иметь в виду, что карты изображений большого размера могут затруднять загрузку для пользователей низкоскоростных интернет-соединений.

Установка параметров экспорта изображений в формате HTML
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий дважды щелкните Документ, а затем выберите Экспорт HTML.
3.
 
Настройте параметры растровых изображений.

Дополнительные возможности
 
Установка количества пикселей, на которое может автоматически смещаться текст, чтобы предотвратить создание строк или столбцов размером всего в несколько пикселей.
Введите значение в поле Допуск по положению.
Установка допустимого количества пикселей в пустой ячейке перед объединением этой ячейки с соседней ячейкой.
Введите значение в поле Пробел изображения.
Это позволяет предотвратить разделение единого графического изображения, занимающего соседние ячейки.
Установка величины пробела, разрешенной для изображения
Введите значение в поле Пробел положения.
Создание карты изображений на сервере
В области Тип карты изображений установите флажок Сервер и выберите формат.
