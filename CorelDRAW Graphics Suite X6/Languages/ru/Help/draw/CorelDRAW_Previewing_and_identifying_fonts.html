Предварительный просмотр и идентификация шрифтов

В CorelDRAW можно выполнять предварительный просмотр шрифтов до их использования. Кроме того, шрифт можно просмотреть в любых имеющихся стилях, например жирным или курсивом.

Можно также легко идентифицировать шрифт, используемый в тексте графического дизайна. Приложение CorelDRAW позволяет выполнить захват образца графического дизайна и автоматически выгрузить его на веб-узел www.whatthefont.com для идентификации.

Идентификация шрифта
1.
 
Выберите пункт Текст WhatTheFont?!
2.
 
Отметьте область со шрифтом, который требуется идентифицировать, с помощью выделяющей рамки. Для выполнения захвата щелкните внутри области захвата или нажмите клавишу Enter. Для отмены процедуры нажмите клавишу Esc.
На веб-узле WhatTheFont?! отобразится захваченный шрифт.
3.
 
Следуйте инструкциям, отображаемым на веб-узле WhatTheFont?!, для завершения процедуры идентификации шрифта.
Наиболее подходящая высота букв, при которой обеспечиваются наилучшие результаты поиска, составляет 100 пикселей. Захватывать следует только буквы верхнего или нижнего регистра, но не цифры и не специальные символы. Строки захватываемого текста должны располагаться горизонтально, а буквы не должны соприкасаться друг с другом.
Предварительный просмотр шрифта
1.
 
Выберите инструмент Текст .
2.
 
На панели свойств щелкните поле Список шрифтов.
Имя шрифта отображается данным шрифтом.
Имена специальных шрифтов отображаются шрифтом интерфейса пользователя по умолчанию с примерами специального шрифта, которые отображаются справа от его имени.
Если шрифт имеет другие стили, например жирный и курсив, то шрифт можно просмотреть, щелкнув раскрывающую стрелку рядом с именем шрифта.
Отображение шрифтов документа
1.
 
Выберите пункт Текст Параметры списка шрифтов.
2.
 
Установите флажок Показывать только шрифты документа.
В списке Шрифт на панели свойств отображаются только шрифты, которые используются в активном рисунке, когда активен инструмент Текст .
Последние использованные шрифты отображаются в верхней части списка Шрифт. По умолчанию в списке отображаются пять последних использованных шрифтов. Можно изменить число шрифтов, отображаемых в этом списке, указав значение в поле Число использованных шрифтов для отображения. Максимальное число последних использованных шрифтов, которое можно задать для отображения — 20.
