Создание объектов для использования в Интернете

CorelDRAW позволяет экспортировать объекты, оптимизированные для просмотра в веб-браузере. Можно также создавать интерактивные ролловеры из объектов CorelDRAW. Кроме того, в рисунок можно добавлять гиперссылки и закладки.

Веб-совместимый объект, созданный в CorelDRAW, можно добавлять в дизайн-проект с помощью веб-инструмента. Также можно экспортировать объект в HTML. Для получения дополнительных сведений см. раздел Экспорт в формате HTML.

В этом разделе рассматриваются следующие темы.

•
 
Экспорт растровых изображений для использования в Интернете
•
 
Сохранение и применение веб-заготовок
•
 
Экспорт объектов с прозрачными цветами и фонами
•
 
Создание ролловеров
•
 
Добавление закладок и гиперссылок в документы
•
 
Добавление горячих точек и альтернативного текста в объекты
