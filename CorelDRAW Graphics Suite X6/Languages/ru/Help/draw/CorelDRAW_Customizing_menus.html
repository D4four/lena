Настройка меню

Функциональные особенности настройки Corel позволяют изменять строку меню и содержащиеся в ней меню. Можно изменять порядок расположения меню и команд меню; добавлять, удалять и переименовывать их; а также добавлять и удалять разделители команд в меню. Существует возможность поиска команды меню, если меню, в котором она находится, точно не известно. Кроме того, можно восстановить для меню параметры по умолчанию.

Параметры настройки используются как для меню, принадлежащих строке меню, так и для контекстных меню, доступ к которым можно получить, щелкнув правой кнопкой мыши.

В разделах справки используются параметры приложения, заданные по умолчанию. При настройке меню и команд меню в разделах справки, связанных с ними, выполненные изменения не отображаются.

Изменение порядка расположения меню и команд меню
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Команды.
3.
 
В окне приложения перетащите меню по строке меню влево или вправо.
Если требуется изменить порядок расположения команд меню, щелкните меню в строке меню, щелкните команду меню и перетащите ее вверх или вниз.
Если требуется изменить порядок расположения команд контекстного меню, щелкните правой кнопкой мыши окно приложения, чтобы отобразить контекстное меню, а затем перетащите команду меню в другое место.
Переименование меню или команды меню
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Команды.
3.
 
Выберите категорию команды в верхнем списке.
4.
 
Щелкните меню или команду в списке.
Черная стрелка рядом с именем в списке указывает, что это меню.
5.
 
Перейдите на вкладку Вид.
6.
 
Введите имя в поле Заголовок.
Амперсанд (&) перед буквой в поле Заголовок указывает на то, что это сочетание клавиш или клавиши быстрого доступа. Меню отобразятся при нажатии клавиши Alt и буквы. Команды активизируются при отображении меню путем нажатия подчеркнутой буквы.
Можно сбросить имя до имени по умолчанию, выбрав пункт Значения по умолчанию.
Добавление или удаление элемента в строке меню
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Команды.
Если требуется удалить элемент, перетащите его со строки меню.
3.
 
Выберите категорию команды в верхнем списке.
4.
 
Перетащите элемент в строку меню.
Черная стрелка рядом с именем в списке указывает, что это меню. Все другие элементы списка являются командами.
Добавление или удаление команды меню
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Команды.
Если требуется удалить команду из меню, щелкните имя этого меню и, когда оно отобразится, перетащите команду за пределы меню.
3.
 
Выберите категорию команды в верхнем списке.
4.
 
Перетащите команду в меню в окне приложения.
Добавление или удаление разделителя команды в меню
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Команды.
Если требуется удалить разделитель команды из меню, щелкните имя этого меню и, когда оно отобразится, перетащите разделитель команды за пределы меню.
3.
 
Выберите пункт Меню пользователя в верхнем списке.
4.
 
Щелкните разделитель и перетащите его на меню в окне приложения.
Быстрый поиск команды меню
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Команды.
3.
 
Нажмите кнопку Поиск .
4.
 
В диалоговом окне Поиск текста в поле Найти введите команду меню.
5.
 
Нажмите кнопку Найти далее.
Восстановление для меню параметров по умолчанию
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий Настройка выберите Панели команд.
3.
 
Выберите в списке Строка меню.
4.
 
Нажмите кнопку Сброс.
