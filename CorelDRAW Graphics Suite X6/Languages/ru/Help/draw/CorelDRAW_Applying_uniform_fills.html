Применение однородных заливок

Для объектов можно применить однородную заливку. Однородные заливки представляют собой сплошные цвета, которые можно выбрать или создать с помощью цветовых моделей и цветовых палитр. Для получения сведений о создании цветов см. раздел Работа с цветом.

Применение однородной заливки
1.
 
Выделите объект.
2.
 
Щелкните цвет в цветовой палитре.
Если необходимо смешать цвета в однородной заливке, нажмите клавишу Ctrl и щелкните другой цвет в цветовой палитре.
Можно также выбрать однородную заливку, щелкнув любой из следующих элементов управления.
•
 
Нажмите кнопку Однородная заливка в области Заливка окна настройки Свойства объекта.
•
 
В наборе инструментов нажмите кнопку Однородная заливка.
•
 
Выберите инструмент Интерактивная заливка в наборе инструментов, затем выберите Однородная заливка в списке Тип заливки на панели свойств.
