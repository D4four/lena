Изменение настроек запуска

Можно изменить параметры запуска CorelDRAW, определяющие вид приложения при запуске. Например, можно запускать приложение с открытым экраном приветствия или новым пустым документом.

Изменение настроек запуска
1.
 
Выберите Инструменты Параметры.
2.
 
В списке категорий Рабочее пространство выберите Общие.
3.
 
В области Начало работы выберите параметры в списке При запуске CorelDRAW.
Если требуется скрыть диалоговое окно Создание документа, которое появляется при создании нового документа, снимите флажок Диалоговое окно «Показать новый документ».
