Работа с символами

CorelDRAW позволяет создавать объекты и сохранять их в качестве символов. Символы определяются один раз, и на них может быть множество ссылок на рисунке. Каждый раз при вставке символа в рисунок создается экземпляр символа. Определения символов, а также сведения об их экземплярах, хранятся в диспетчере символов, который является частью файла CorelDRAW (CDR). Использование символов для объектов, которые отображаются в рисунке множество раз, способствует уменьшению размера файла.
