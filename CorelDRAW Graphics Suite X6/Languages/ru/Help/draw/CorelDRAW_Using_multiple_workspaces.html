Использование нескольких рабочих пространств

Рабочее пространство — это конфигурация параметров, определяющая способ расположения различных панелей команд, команд и кнопок при открытии приложения. Рабочие пространства можно создавать и удалять, а также можно выбрать готовое рабочее пространство программы. Кроме того, для текущего рабочего пространства можно восстановить параметры по умолчанию.

Можно выполнять экспорт рабочих пространств на другие компьютеры, использующие то же приложение, и их импорт с этих компьютеров. Например, можно настроить рабочее пространство и использовать его совместно с другими пользователями.

Создание рабочего пространства
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий щелкните Рабочее пространство.
3.
 
Щелкните Создать.
4.
 
Введите имя рабочего пространства в поле Имя нового рабочего пространства.
5.
 
В списке Использовать для создания нового рабочего пространства выберите существующее рабочее пространство, на основе которого будет создано новое.
Если требуется включить описание рабочего пространства, введите описание в поле Описание нового рабочего пространства.
Специальные рабочие пространства сохраняются как файлы XML и экспортируются как файлы формата Extensible Stylesheet Language Transformations (XSLT) на основе XML.
Выбор рабочего пространства
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий щелкните Рабочее пространство.
3.
 
Установите флажок рядом с выбранным рабочим пространством в списке Рабочее пространство.
Если необходимо выбрать рабочее пространство во время запуска приложения, убедитесь, что установлен флажок Выбор рабочего пространства при запуске.
Удаление рабочего пространства
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий щелкните Рабочее пространство.
3.
 
Выберите рабочее пространство в списке Рабочее пространство.
4.
 
Нажмите кнопку Удалить.
Нельзя удалить рабочее пространство, заданное по умолчанию.
Восстановление текущего рабочего пространства
1.
 
Закройте окно приложения.
2.
 
Повторно запустите приложение, удерживая нажатой клавишу F8.
Импорт рабочего пространства
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий щелкните Рабочее пространство.
3.
 
Нажмите кнопку Импорт.
4.
 
В диалоговом окне Импорт рабочего пространства нажмите кнопку Обзор.
5.
 
Выберите папку, в которой хранится файл.
6.
 
Дважды щелкните файл.
7.
 
Следуйте инструкциям на экране.
Экспорт рабочего пространства
1.
 
Выберите пункт Инструменты Настройка.
2.
 
В списке категорий щелкните Рабочее пространство.
3.
 
Нажмите кнопку Экспорт.
4.
 
Установите флажки рядом с элементами рабочего пространства, которые требуется экспортировать.
5.
 
Нажмите кнопку Сохранить.
6.
 
Выберите папку, в которой необходимо сохранить файл.
7.
 
В поле Имя файла введите имя файла.
8.
 
Нажмите кнопку Сохранить.
9.
 
Нажмите кнопку Закрыть.
Элементами рабочего пространства, доступными для экспорта, являются окна настройки, панели инструментов (включая панель свойств и набор инструментов), меню, строка состояния и сочетания клавиш.
Специальные рабочие пространства экспортируются как файлы формата Extensible Stylesheet Language Transformations (XSLT) на основе XML. Можно использовать рабочее пространство, сохраненное в формате XSLT, при установке или использовании специальных рабочих пространств.
Можно отправить рабочее пространство как вложение по электронной почте, щелкнув Email в диалоговом окне Экспорт рабочего пространства.
