Настройка дополнительных параметров штрихкода

Для штрихкода можно указать дополнительные параметры, например задать контрольные цифры или изменить внешний вид штрих-кода. Контрольная цифра определяет правильность считывания данных. Закодированные цифры подставляются в определенную формулу, и получается контрольная цифра. Контрольная цифра обычно стоит в конце штрихкода. Компьютер проверяет правильность считывания цифр, сравнивая рассчитанную контрольную цифру со считанной.

С помощью других дополнительных параметров можно изменить внешний вид символа. Используя дополнительные параметры, можно, например, добавить к штрих-коду числовой префикс и задать игнорирование интервалов и скобок в символе.

Настройка дополнительных параметров
1.
 
Нажмите кнопку Дополнительно на странице Стандартные отраслевые свойства.
2.
 
Укажите параметры, которые требуется применить.
Добавлять или удалять контрольные цифры, скобки, интервалы и префиксы можно только для следующих штрих-кодов: Codabar, Code 39, Code 128, ISBN, ISSN, ITF-14, MSI Plessey и UPC(A).
